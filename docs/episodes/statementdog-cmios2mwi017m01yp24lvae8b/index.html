提及公司

三句話總結

  • 美國基金經理人Q3整體微幅減碼：整體倉位仍集中科技，減碼邏輯像是「漲太多」後的調整，而非看壞市場；Google人投數減碼最多，但按重倉計仍是最大持股。
  • 產業佈局偏多方向：啤酒（估值低）、軟體SaaS（Salesforce/Adobe防守AI衝擊）、能源（天然氣出口+油氣週期底部）、鐵路（UNP+NSC整併題材）、區域銀行（監管放鬆+便宜）。
  • FISV大洗澡後浮現機會：新管理層上任六個月後大幅下調預期（Revenue從9-12%降至持平，EPS從2.6降至2.0），股價當日跌40%；競爭優勢未受損，下調後本益比僅7-8倍，13F季度顯示爆雷前已有大量機構低位買入。

投資觀點

  • 年底稅損賣壓：今年跌較多的個股年底面臨稅損了結壓力，可等12月下半月賣壓出盡再進場（FISV即為典型案例）。
  • 高品質公司估值修正後是機會：FISV屬於支付科技寡占市場（銀行端客戶留存率99%），此次為新管理層洗澡式下調，非競爭力受損。
  • 鐵路整併邏輯清晰：UNP+NSC一旦合併，形成美國第一條東西向跨大陸鐵路；波克夏BNSF+CSX整併亦可預期；CNI因無適合並購對象而股價最弱，但有關稅受益論點支撐。

美國基金經理人Q3 13F分析

整體倉位

  • 普遍微幅減碼，但整體仍集中科技；Google人投數減碼最多，但重倉仍是最大持股。
  • 邏輯為「漲多後的調整」，而非看壞市場。
  • 預期年底稅損賣壓：今年表現不佳的個股（如FISV跌40%）將面臨進一步賣壓。

啤酒產業

  • Heineken、Modelo（SDZ）等啤酒股多家經理人同時買入，唯獨Anheuser-Busch（百威）無人買。
  • 估值已修正至10-15倍（長期低位）；有看法認為2025年葡萄豐收年，利潤率改善。
  • 年輕人飲酒比例下降是共識，但調查基準定義有爭議；整體看不出強烈的買入主題。

軟體SaaS

  • AI衝擊論修正中：Salesforce整合Data Cloud（企業資料不外洩，在平台內用AI分析）；Adobe強調版權安全（從Adobe平台生成的內容無版權疑慮）。
  • 整體估值大幅下修後吸引資金進入；Adobe本益比顯著低於Salesforce，但Salesforce整合邏輯更直觀。

能源（油氣）

  • 天然氣：2026-2030年LNG出口長期合約帶動，EQT為主要受益者。
  • 石油：CapEx過低，以現有技術僅維持現有產量已需加資本支出；部分機構認為明年中為週期底部，XOM為代表。
  • 核能受惠AI缺電主題但反應過快；核電廠重啟僅能提供3-4GW，遠不及2030年資料中心新增40-50GW需求，短期主要仍靠天然氣。

鐵路整併

  • UNP + NSC：若合併，形成美國第一條東西向跨大陸鐵路，大幅降低貨運成本（目前需分段換乘）。
  • BNSF + CSX：UNP+NSC若過關，波克夏BNSF預期跟進併CSX；共和黨執政期間對併購態度寬鬆，有時間窗口。
  • CNI：T型鐵路（加拿大東西向+美國中部），無適合並購對象；股價最弱，但關稅議題下海運轉陸運邏輯支撐有部分看多聲音。
  • 美國貨運80%仍由卡車承擔；鐵路整合後競爭優勢在於直通東西岸降低中轉時間與成本。

區域銀行

  • 2023年矽谷銀行倒閉後監管要求大幅提高資本充足率；現在共和黨執政方向是放鬆，原本要求增加19%的資本金可能降至9%或以下。
  • 資本釋放空間可用於回購與股利，但主要受益是大型銀行（資本要求最高的是系統重要性銀行）。
  • 區域銀行選股：TFC（整併調整期，利潤率有提升空間）、MTB（保守經營、估值低）。
  • AI導入與數位化屬固定成本，驅動區域銀行持續整併以攤提成本。

個股深度：FISV（Fiserv）

  • 業務：支付科技解決方案，唯一同時服務商家端與銀行端的公司；銀行端客戶留存率99%，屬高品質寡占業務。
  • 爆雷事件：新管理層上任六個月後大洗澡——Revenue指引從9-12%下調至持平，EPS從2.6降至2.0（阿根廷業務今年表現佳，不可持續）；股價單日跌40%。
  • 管理層解釋：前任管理層做過多盈餘管理（加一次性費用、削減資本支出），損害長期競爭力；新管理層逆轉政策，競爭力應未受損。
  • 當前估值：下調後EPS的7-8倍本益比，屬高品質公司中罕見的低估值；13F顯示爆雷前已有多家機構建立小倉位。
  • 進場策略：等12月下半月稅損賣壓出盡後再進場，預計下季13F買盤將大幅增加。

節目金句

  • 「能源短期是在週期下行，可是大家在猜週期底，這是從來都不只一兩家在講的事情了。」
