中東地緣政治深度分析

霍爾木茲海峽「收費戰模式」

  • Sitny研究報告實地調查：分析師親赴海峽現場（飛抵杜拜→開車到海峽→搭民間船出海），發現伊朗並非全面封鎖，而是採「準海關制度」——依國家差別管控。
    • 中國船隻：免費通行（以凍結伊朗海外資產和政治讓步換取通行）
    • 其他國家：需繳費或外交換機後取得通行許可碼，由革命衛隊引導通過特定水道
    • 觀察員船隻：因無大量貨物，伊朗衛隊放行
  • 含義：全面斷航對各方均為毀滅性打擊，雙方存在默契保留航路；只要有船持續通過，市場願意重新定價。

伊朗軍事與能源雙重策略

  • 拼圖式防禦：指揮權分散至各地部隊，通訊受阻時用光纖電話線維持；移動發射車打完就轉移，難以追蹤摧毀；無法以正面空戰取勝，改以消耗戰拖延。
  • 石油收入不降反升：3月出口維持240-280萬桶/日；油價飆升使收入翻倍；90%出口至中國地方煉油廠，透過影子公司和走私掩蓋來源。
  • 胡賽武裝籌碼：尚未封鎖阿曼/鴻海（霍爾木茲出口方向）；伊朗將胡賽視為談判籌碼，只要葉門財政破產就需伊朗支持才能升級攻擊。
