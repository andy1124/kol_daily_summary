節目：股癌 日期：2025-12-24

提及公司

  • 台股：AP Memory（矽成半導體）
  • 其他：TSM（台積電先進封裝），Speал（外部 Interposer）

三句話總結

  • 2025 年績效總結：艾德今年表現略優於去年，Tesla 及小鬼谷部位年底意外全醒，整體靠動能策略與小型股長抱策略雙輪驅動。
  • 投資策略演進：4 月互惠關稅事件後更積極管控流動性風險，槓桿從 1.7 倍壓縮至 1.3-1.5 倍，並增設多管道備用入金管道，兼顧部位安全與心理舒適。
  • AP Memory 技術觀察：VHBM（Very High Bandwidth Memory）結合 IPC/IPT 電容方案有機會受惠於 CoWoS 擴張；良率達 7X% 後市場關注點轉向能否獲大客戶（博通等）量產訂單。

投資觀點

  • 槓桿管理：現維持約 1.3-1.5 倍，同業中屬極度保守；4 月跌停潮後認識到帳戶間流動性分配不均的系統性風險，解方是「各帳戶維持率拉平 + 場外備用資金管道」。
    • 除券商融資外，已談妥銀行與租賃公司備用資金管道（低利率）。
  • 小鬼谷（小型股長抱）策略：鎖定台股市值 50-70 億以下標的，以「買入並持有」為主，買入理由消失才出場；今年年底前表現轉佳，計劃持續加大此部位比重。
  • 美股 vs. 台股：美股以動能 + 長抱為主，接受大波動換取爆擊機率；台股小型股需注意流動性（出不來的問題），限制了可投入上限。
  • 出場條件優先於進場條件：「買進理由不見了就出場」——如散熱股因 Immersion Cooling 死掉而砍出，消費晶片股因主力作動而非消費復甦噴出時順勢出場。

AP Memory 技術分析

  • IPC（放在矽中介層的電容）：主要用於 CoWoS 封裝，需跟上台積電或 Speal 外部 Interposer 產能；良率達 7X% 為關鍵門檻。
  • IPT（宅板內電容）：隨 CoWoS-L 擴大有望進入宅板供應鏈；能否從 IP 授權轉為量產貢獻營收尚不確定。
  • VHBM：主要客戶為 Meta（嚴禁），此前為礦機；搭配 Weft-on-Weft（WoW）堆疊技術，需求增長中。
