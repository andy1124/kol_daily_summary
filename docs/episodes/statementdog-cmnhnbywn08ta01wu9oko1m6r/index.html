513.【財經時事放大鏡】光油並進

節目：財報狗 Podcast　｜　日期：2026-04-02

提及公司

  • 美股：NVIDIA、Marvell、Broadcom（Boccon）、Coherent、Lumentum、Intel
  • 台股：中油、台塑（未具名塑化廠）、光通訊相關供應鏈（未具名）
  • CSP：Google、AWS、Meta、Microsoft

三句話總結

  • 中東衝突重塑石化供應鏈：荷莫茲海峽限制通行、石化原料缺貨漲價，台灣塑化廠長期壓制的競爭劣勢有機會翻轉。
  • NVIDIA 投資 Marvell 戰略意圖：將 Marvell 拉入 NVIDIA Link Fusion 生態系，確保 ASIC 時代仍可透過光通、封裝、CPU 切入各 CSP 基礎建設。
  • 光通 Scale-up 量爆在即：800G 模組已供不應求，2026-2027 年光通 Scale-up Network 成最大成長引擎，但技術路線細節仍在演進中。

投資觀點

  • 石化/塑化：存貨利益窗口打開：低價庫存遇原料漲價，近期利差擴大；只要戰爭持續、伊朗供貨受阻，台灣塑化廠的價格話語權回升，但需觀察能否「常態化收費」。

  • 光通仍是主線：800G 模組現貨已無，只有期貨可買；26 年想購買光模組需要提前鎖定產能。估值高但 Scale-up 量爆撐起長期想像。

  • NVIDIA 生態系擴張邏輯：越來越多 CSP 自研 ASIC，NVIDIA 選擇「加入而非阻止」——透過 NVIDIA Link Fusion 確保非 GPU 架構的資料中心仍使用其 Switch、光通、CPU 等元件。

中東衝突與石化供應鏈

荷莫茲海峽影響

  • 通行正常化為付費模式：伊朗目前對非美國籍船隻收費放行，類似巴拿馬運河效應。
  • 供給縮減結構：每天 50 萬桶通過，美國航空燃油出口量僅 21 萬桶，無法完全補位；日本等國積極與伊朗談判。
  • 台灣石化依賴度：台灣約七到八成石油來自荷莫茲海峽附近，另外兩成來自美國（川普邏輯：你們不用繞道，直接向我買）。

塑化廠受惠邏輯

  • 中國打壓源頭：過去伊朗廉價供油讓中國塑化廠成本極低，台廠被虐殺；若伊朗供貨中斷，台廠競爭力回升。
  • 存貨利益：低價進料庫存遇漲價環境，可認列存貨利益；目前早料市場「一天數個價格」。
  • 時間軸：至少持續三年以上的供應重組，非短期事件題材。

NVIDIA × Marvell 戰略解析

投資意圖

  • NVIDIA 投資 Marvell 兩個 B：將 Marvell 拉入 NVIDIA Link Fusion（NV Fusion）生態系，具體包含：

    • Marvell 的 XPU 可配合 NVIDIA Link 使用其 IP
    • 自研 ASIC 的 CSP 可透過 Marvell 取得 NV Fusion 小 Chiplet 搭配封裝
    • 擴大光通 Scale-up 陣營：Coherent/Lumentum 負責雷射端，Marvell 負責設計端

  • 對 Broadcom 的影響：

    • 以往 ASIC 設計唯一重量選擇是 Broadcom 或 Marvell；現在多了 NVIDIA 選擇
    • Broadcom 相對處於弱勢，但市場盤子夠大，不是零和遊戲

  • 生態系軍火庫邏輯：類比台積電——不管誰的 ASIC，連接、封裝、CPU 都要靠 NVIDIA；NVIDIA 的「軍火庫」策略讓其在 ASIC 時代不被邊緣化。

Marvell 角色

  • Marvell 本身已承接大量 CSP 的 ASIC Design Service；加入 NV Fusion 後相當於多一個背書，可望吸引更多 ASIC 設計案。
  • Marvell 子公司 Inphi（已被收購）與 CXL/PCIe Switch 解決方案組合，理論上可做到 XPU 間的記憶體池化（Memory Pooling）。

光通 Scale-up：下一個量爆點

現況

  • 800G 光模組：供不應求，想買只能排期貨（明年下半年起）；EML 雷射等核心元件幾乎全線滿產。
  • 需求來源：Scale-up Network 是光通最大 Upside，Switch IC 數量乘以 Optical Engine 數量已進入億級規模。

技術路線演進

  • DFB → VCSEL → EML：短距光通逐步走向 Co-Packaged Optics（CPO），Scale-up Network 是最大量爆場景。
  • Strelation 概念：在晶片後面嵌入光通道，美好願景是將所有加速器透過光纖全部聯結——類似「矽光子京城」。
  • LED 光通（短距）：屬新創/A 輪階段，便宜但良率挑戰大，尚不在主流討論範圍。

成本門檻

  • 每 Gigabyte 傳輸成本：已接近銅線同等水準（約 $3.26 美元），差一步即可引爆大規模替代。
  • 量產時間：一旦量產放量，成本曲線快速下降；目前各廠商都知道很賺，正積極擴產。

石化/塑化
