投資觀點

  • 硬體vs.軟體的結構性分化仍在持續：NDX與IGV（軟體ETF）從2025年下半年起全面脫鉤，市場定義AI超級循環等同硬體，消費、軟體、能源被抽離資金。

    • IGV昨天大跌近3%，仍處於超低人氣的「擠壓反彈」階段而非基本面改善
    • 長期而言NDX與IGV高度連動，若AI商業模式走向落地，軟體股可能重新成為輪動目標

  • 台積電條款改變主動型ETF的配置邏輯：

    • 過去ETF被迫買0052或0050來間接增加台積電敞口（疊床架屋），現在直接買台積電即可
    • 主動型ETF今年平均漲43%，三檔（00992A、00981A、00994A）漲幅突破60%，資金高度集中AI相關產業
    • 持股上限從10%→25%，系統性回補可能帶動台積電成為全台ETF績效的共同驅動力

  • 外資的角色：不回補台股照漲，回補了漲更快：2020至2026年外資每年均為賣超，台股仍從8,000點漲至近4萬點，根本動力來自外銷訂單轉化的外匯收入，再流入內資買盤。

  • 能源股（XLE）開始止跌回穩：油價重回每桶100美元，地緣政治風險上行，能源股從觀望轉多，若未來進入新景氣上行，油價80至100美元的「新常態」將帶來獲利拉升。

昨日盤面解讀（2026/4/23）

  • 上市+櫃買成交量達1.3兆（歷史高），貴買佔交易量約30%，波動大但非換手而是換股。
  • 外資賣超僅27億（幅度小），主要是內資中實戶與散戶同時獲利了結，引發多殺多。
  • 台積電仍強勢未崩，中小型股受資金擠壓跌幅放大，為典型「換股行情」而非趨勢反轉。
  • 1,000點內的波動、8,000億內的量能屬正常，台股身為全球第六大市場，需給予更寬的容忍度。

主動型ETF的競爭邏輯

  • 投信持有台積電上限10%→25%：不調高持股即績效脫鉤大盤，主動型經理人有強烈動機率先回補。
  • 各ETF過去因法規無法超配台積電，被迫購買0052/0050作為曲線持有台積電的方式，現在枷鎖解開。
  • 資金邏輯：ETF管理規模（AUM）計算，若全體投信從10%拉至25%，需回補台積電數萬至十萬張。

  • 台灣薪資：3月平均薪資含年中獎破9萬元（月增近三成），中位數約3.9至4萬元，差距仍大；金融保險、資通訊業月薪6至7.4萬，服務業、餐飲業3至4萬出頭，缺工現象嚴重。

資金配置原則

  • 選大、選AI方向：大股票（SOXX、QQ、0050、台積電）報酬遠優於中小型個股追題材；中小型股多殺多時首當其衝。
  • 軟硬體通吃：持有SOXX或QQ者不論費半或納指走強均受益，避免押注單一敘事。
  • 比特幣觀察：比特幣卡在長期下降趨勢線和100日均線，有效突破7.5至8萬才考慮介入。
  • 特斯拉謹慎：EPS連續四季低於市場預期（Q1每股盈餘0.25美元vs.市場預期0.6美元），散戶每日資金70億美元撐住支撐，但追高意願薄弱，壓力仍大。
