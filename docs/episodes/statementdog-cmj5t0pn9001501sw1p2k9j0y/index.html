494.【達人聊產業】HVDC 來臨，AI、儲能 & 電動車 ft. 松川吳頌仁副董事長

日期：2025-12-14

提及公司

  • 美股：NVIDIA、Tesla（電動車 Model 3/Y、Megapack儲能）、GE Vernova（渦輪發電機）、Mercedes-Benz（賓士 EQ/PHEV）、Porsche（Taycan 800V電動車）
  • 台股：松川精密（繼電器廠，2025年新上市）、台達電、光寶科技、廣達、鴻海
  • 其他：寧德時代（中國，電池）、陽光電源（中國，儲能）

三句話總結

  • HVDC架構轉換讓繼電器量少、單價大漲：傳統AC PSU一機架需多達36顆繼電器，HVDC架構下一台DC PSU只需1-2顆，但因高壓DC技術難度高（需解決電弧問題），單價大幅提升，松川以「非真空密閉型」方案切入，毛利較佳。
  • 稀土管制倒逼三路並行：重稀土被管制後，松川同步推進廈門廠就近採購磁鐵組裝、輕稀土替代設計、及長期的無稀土（鐵氧體）方案，保住賓士等歐美重要客戶。
  • 儲能業務上半年年成長109%：電池成本大降讓潛在剛性需求爆發；松川客戶涵蓋全球最大電站式儲能商（Tesla Megapack + 陽光電源），表前（電網級）儲能仍是最確定的成長驅動。

投資觀點

  • 松川 HVDC 卡位早、技術驗證完整：2011年即開始研發非真空密閉型高壓DC繼電器，賓士PHEV已採用多年；相較傳統真空封裝成本高，松川方案能壓低成本又維持效能，進入下一波AI伺服器HVDC架構有先行優勢。
  • 繼電器受益 AI Data Center 電力升級：算力每代提升→PSU功率從commodity的5.5kW往更高推進，電壓從AC 240V走向HVDC 400V/800V；電流、電壓雙升對繼電器（接觸器）技術要求與單價均大幅提升。
  • 儲能「表前」為最剛性的需求：不受利率高低影響，全球新增再生能源佔比達80%（2024），儲能搭配成為電網必需；表後（家用）仍受電費差與利率影響，分化明顯。
  • 電動車族群：400V vs 800V 取決設計思維而非成本：Tesla Model 3/Y仍走400V（產業鏈成熟、成本控制），Porsche Taycan走800V（快充體驗優先）；市場滲透率往25%推進，松川電動車佔比高達38%，為最大單一應用。

松川精密 / 繼電器產業深度分析

公司背景與轉型歷程

  • 成立於1974年，台灣第一家繼電器廠，早期以家電用途為主（現僅剩8%）。
  • 2010年啟動轉型：聚焦高壓DC、高電流、節能繼電器三條技術路線；2016年前後新能源應用佔比快速提升。
  • 2025年產品比重：家電8%、AI電源12%、傳統汽車21%、電動車38%、儲能/太陽能21%；目標從60%新能源拉高至70%以上。

繼電器技術壁壘

  • 繼電器 vs 接觸器：30A以下稱繼電器，以上稱接觸器，但技術邊界已模糊，松川已跨入1000V AC / 800V DC域。
  • 高壓DC電弧問題：AC電流過零點自然熄弧；DC無此特性，電壓拉高後接點開關如同電焊，需特殊解法。
  • 松川非真空密閉型方案：相較傳統真空陶瓷封裝成本更低，已通過賓士PHEV認證多年；移植至AI伺服器HVDC場景，目標縮小至1U尺寸內。

稀土管制應對

  • 重稀土（HRE） 用於節能繼電器（Latching Relay）中的永久磁鐵，耐溫/磁力特性優異但受中國管制；台廠申請許可更難取得。
  • 三路並行方案：①廈門廠就近採購→組裝後出口（已執行，大半年努力）；②輕稀土替代設計（重新跑設計驗證）；③無稀土鐵氧體（磁能積小，需更大體積，技術挑戰高，長期解法）。

HVDC 對松川影響

  • 傳統 AC PSU：每機架可達36台PSU，每台多顆繼電器，為量大但單價低的 commodity。
  • HVDC DC PSU：轉換級數大減（效率提升45%），每台PSU僅需1-2顆繼電器，但技術難度高、單價倍增，毛利率顯著優於傳統。
  • 明年底/後年才正式大量出貨HVDC PSU；目前仍以AC PSU為主，HVDC為下一個成長曲線。

儲能市場分析

全球爆發背景

  • 電池成本大降 讓潛在剛性需求爆發；全球2024年新增發電容量80%為低碳能源，再生能源比重高→電網韌性需儲能配套。
  • 松川儲能業務H1 YoY+109%；主要市場：美國、澳洲、歐洲、日本、部分中東東南亞，台灣比重相對小。

分類與機會

  • 表前（電站式）：剛性需求最強，電力公司/資料中心旁邊自建儲能，不受利率影響。
  • 表後（家用/商用）：受電費差、利率、居住型態影響，成長較慢，台灣因電費低而普及率低。
  • 移動式儲能（露營/緊急備電）：消費性，受景氣利率影響。

節目金句

  • 「我們在2010年的時候都還不知道有這些東西，之所以能夠掌握這些機會，是因為我們已經擁有這個技術了。」
  • 「電動車的用戶問他會不會想回去再開汽油車，答案應該不會——這是一個非常重要的趨勢。」
  • 「高壓直流的電弧就像電焊——你的接點一下就融掉了，所以在高電壓直流上面也要用到磁鐵。」

  • 「我們從2011年就開始研發非真空密閉型的高壓DC繼電器，這項技術已得到賓士的認可，採用好多年了。」
