AI 重點摘要

提及公司

  • 美股：SpaceX、Tesla、ASTS（AST SpaceMobile）、Amazon（Project Kuiper / TeraWave）、Robolox（提及 IPO 類比）
  • 台股：旺宏（Winbond，作為警示案例）、台積電、華通（衛星 PCB）、2367（聯鈞，衛星 PCB）、MACO（日本小市值公司）、博士（PCB 廠）
  • 其他：MagicLep（Macle，提及越南合資）

三句話總結

  • 市場廣度全面打開，廠務股/工具股全面啟動：關稅 15% 救活大量原本躺平的傳統產業股，被動元件、ABF 基板、記憶體漲價齊發，連網通廠都成功把成本轉嫁出去。
  • 衛星供應鏈進入 Bottom-up 驗證階段：不僅有 Top-down 訂單線索，M6 衛星板、PCBA 的具體料號已在台灣廠商出貨清單中確認，華通、2367 等衛星 PCB 廠受惠邏輯更加清晰。
  • SpaceX + Amazon TeraWave 加速，台灣供應鏈第二波想像空間：Bezos 推出 6 Tbps 企業級衛星服務 TeraWave，加上 Starlink V3/V4 持續射衛，衛星板/PCBA/特殊材料進入缺貨漲價預期。

投資觀點

  • 廠務股（工具股）全面啟動是關鍵訊號：以前在地上躺的東西，關稅帶來訂單轉移後開始動起來，代表市場廣度確認，不只是少數 AI 龍頭的獨舞。
  • 原物料漲價傳導路徑清楚：銀價上漲 → 使用銀的被動元件成本上升 → 被動元件廠轉嫁 → 即便銀價後來回落，成品價也難下來（粘性）
    • 網通廠毛利率低（十幾%），不可能吸收記憶體漲價 → 已成功全數轉嫁，B 端客戶接受
    • 接標案的網通廠才有轉嫁壓力，B2B 訂單廠商問題不大
  • 注意營收佔比，而非只看有無題材：旺宏（Winbond）產品到處都有用到，但佔比太分散，股價曾長期不動；要找的是：某產品出貨在特定廠商的營收佔比夠高的標的。
  • 率先動的才是最終漲最多的：衛星概念中華通是率先動的，後面才輪到其他被拉起來的；率先動的標的往往有真實訂單支撐，後補的題材股風險較高。

市場整體分析

廣度擴張

  • 台股廣度全面打開：被動元件、記憶體、ABF 基板、廠務股、衛星板均動
  • 關稅 15% 產生訂單轉移效果，本來躺平的傳統製造業開始有感覺
  • 韓國嘗試修改美軍對台協議 → 川普直接加關稅警告，台灣需謹慎維持關係

漲價結構分析

  • 需求驅動型（主委偏愛）：衛星、AI server 等有實際增量市場的
  • 成本驅動型（也值得關注）：銀漲→被動元件、記憶體漲→網通成本

衛星供應鏈深度解析

Top-down → Bottom-up 雙重確認

  • 之前：頂部推理（看到訂單可能性） → 尋找供應鏈
  • 現在：底部確認（M6 衛星板料號已在廠商出貨清單中出現，且放量規模大）
  • 邏輯更完整：不只是有題材，訂單真的存在

主要玩家與台灣受益廠商

  • SpaceX Starlink：已確認，供應鏈廠商在東南亞（越南、泰國）有生產基地
  • ASTS（AST SpaceMobile）：另一個正在追蹤的衛星公司
  • Amazon TeraWave / Project Kuiper：Bezos 推出 6 Tbps 高速企業級衛星服務，目標 10 萬企業客戶 + 資料中心，結合低軌（LEO）+ 中高軌（MEO）衛星
    • 類比：Starlink = 給消費者，TeraWave = 給企業/資料中心（更高規格）
  • 台灣衛星 PCB 廠：華通（率先動，最具真實訂單支撐）、2367（聯鈞，同為衛星 PCB）
    • 若華通產能滿了，訂單可能外溢到 2367
  • MACO（日本）：小市值，Guidance 極保守，但已看到上游廠商和客戶規模遠大於指引，有超額成長潛力
  • MagicLep（Macle）：在越南與博士合資工廠，官網寫 AI server 用途，但市場認為可能轉向衛星；主委提醒：企業計畫常隨市場最熱題材調整

衛星軌道 AI 算力（更遠期，但已有人推進）

  • 在太空設置 AI 資料中心：無限太陽能、散熱透過輻射方式
  • 目前難題：輻射保護、發電穩定性 → 但 AI 時代大家更勇於解決難題
  • TeraWave 已是邁向企業 AI 計算邊緣的一步

操作策略

衛星概念

  • 找到核心標的後，觀察延伸：誰與誰有合資/投資關係、誰的產能是滿的（找同類替代廠）
  • 等待好位置：衛星股容易被炒進處置，被洗掉前要先找到好的買入點
  • 不要追在最高點，處置後的非理性殺盤才是布局時機

部分減碼訊號

  • 短線題材股：出現大黑棒、把前幾個收盤價都打破 → 減碼訊號
  • 長線持股（數字確認，未來幾年成長清晰）：不用用短線邏輯看
  • 動能型（衛星、設備）：可搭配月線/週線做參考，突破失敗才考慮減

記憶體

  • 持續追蹤：漲價已確立，長期合約談定，下行空間有限
  • 現貨鬆動是套利行為，不要過度解讀

QA

被動元件漲價是否會持續

  • 情境：有聽眾問 B2B 網通廠記憶體漲價轉嫁問題。
  • 重點回答：毛利率 10-13% 的廠商不可能吸收，B 端客戶已接受轉嫁，重點是區分客戶是 B 端還是消費者端（後者轉嫁困難，前者已成功轉嫁）。
