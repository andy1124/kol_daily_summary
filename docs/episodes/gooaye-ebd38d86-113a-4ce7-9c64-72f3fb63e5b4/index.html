三句話總結

  • Intel EMIB 良率突破：Intel 後段封裝良率已達 90%，市場原本悲觀預期落空，短期可視為專業後段封裝廠，有望承接更多 CoWoS 訂單。
  • 記憶體與被動元件表態：旺紅法說轉為超級利多，MLCC/NOR 漲價明確，台股被動元件族群跟上日韓先行漲勢，族群集體補漲。
  • 台積電外資賣壓制約輪動：台積電衝至 2330 後遭外資連日大賣，資金輪動至中小型股及記憶體，盤面進入類股帶開格局。

投資觀點

  • Intel 後段封裝不可忽視：良率 90% 已可量產，市場普遍以載板端 30-40% 良率做錯誤推估，此一資訊落差值得關注。AP memory 為 Intel Substrate 的 IPD 供應商，關聯標的也開始表態。
  • 記憶體基本面無虞，股價等候反映：各方驗證均指向缺貨持續，蘋果法說直接點名後續季度記憶體成本仍將上漲，基本面領先股價。
  • 台積電外資壓制短期看不止：外資一天賣逾萬八、兩萬張，本波台積電若動能無法繼續，資金將持續在記憶體、被動元件、中小型股之間輪動。
  • CPU 熱度確認：Meta 簽下 Multi-Billion Dollar Graviton 5 協議，估算 10s of millions Graviton Core 需求，台系 CPU 供應鏈全面動起。

主題分析區塊

Intel EMIB 封裝良率

  • 90% 良率已可量產：市場普遍透過台系載板廠確認良率只有 30-40%，但載板良率 ≠ Intel 封裝良率，兩者為不同製程段。
  • 類比 Blackwell 初期：Blackwell 初期良率約 40-50%，之後快速 ramp；Intel Substrate 預計也將跟上，後段封裝良率持續拉升有望。
  • 潛在訂單效應：若 Intel EMIB 成功量產，MediaTek（Google TPU 後段備份方案）、AP memory（IPD 供應商）均有關聯受益機會。
  • 前段仍靠台積電：5921（HUMUFISH）前段在台積電，後段規劃一條走 Intel EMiB，一條保留 CoWoS；此事不確定性仍高，屬持續觀察項目。

Google TPU（5921 / HUMUFISH）與台積電封裝

  • 5921 CoWoS 產能：V8/V9 TPU，初始 40K，後年目標拉至 150-180K 之間（眾說紛紜）。
  • 5922 規劃尚早：5921 都還沒大放量，現階段討論 5922 過早，但後續備份方案往台積電移動的概率上升（前台積封裝大將加入聯發科一事的暗示）。

記憶體與被動元件族群

  • 旺紅法說超級利多：MLCC、NOR 漲價明確，老闆態度從謹慎轉強硬，市場解讀為後段確認缺貨。
  • 被動元件補漲：日系村田、韓系三星（被動元件）早已先行漲，台灣族群在外資輪動壓力解除後跟進。
  • 記憶體排擠效應觀察：Intel 已明確表示將產能優先給伺服器，消費級記憶體（DDR3/DDR4、MLC NAND）有排擠性漲價預期，但需求端確認仍待觀察。

CPU 需求確認

  • Meta Graviton 5 大單：Multi-Billion Dollar 協議，推估需求約 52K 顆晶片（以 10M Cores / 192 cores per chip 計算），市場資金跟進買台系 CPU 供應鏈。
  • 消費性 CPU 排擠故事：部分資金押注伺服器 CPU 排擠消費性產能，但 DDR3/DDR4 主要為伺服器規格，消費性量爆難以完全比照記憶體模式，需審慎評估。

OpenAI 智慧裝置

  • AI Native Phone：OpenAI 攜手 li-shin，供應鏈採用高通 + 聯發科雙料晶片，每顆約 $120 美元，市場已有人開始換算 EPS 貢獻。
  • 觀察謹慎：前有 AI PIN 失敗案例，使用習慣改變難度高；li-shin 品質對標 Pixel / Samsung 尚有距離，不宜過度樂觀定價。

類股輪動判斷

  • 台積電失動能後的資金去向：外資持續賣壓 → 記憶體、被動元件表態；若台積電再次發動，中小型將再度承壓，需動態調整。
