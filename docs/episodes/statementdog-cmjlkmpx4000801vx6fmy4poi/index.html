496.【財經時事放大鏡】太空 x 2025 LLM發展 x Groq

節目：財報狗 Podcast
日期：2025-12-25

提及公司

  • 美股：SpaceX（Starlink）、Google（持有SpaceX約10%股權）、NVIDIA（輝達）、Groq、OpenAI、Amazon、Microsoft、Tesla（FSD/自駕）
  • 台股：台灣衛星類股（低軌衛星供應鏈廠商，天線、模組為主）

三句話總結

  • SpaceX IPO帶動低軌衛星全面走強：Starlink已達800萬用戶、遍布150國，2026年營收估值220-240億美元，SpaceX上市預期讓整條太空供應鏈（天線、模組、太空履歷認證廠）全面拉升。
  • LLM效能過剩隱憂初現：Gemini Flash以GPT-5.2三分之一的價格、三倍速度達到同等輸出品質；市場已開始選便宜夠用的模型，業者需重新思考繼續押注「最強模型」是否仍能創造護城河。
  • NVIDIA收購Groq技術資源：Groq的LPU（Language Processing Unit）速度達每秒500+ tokens（OpenAI約50），NVIDIA以近200億美元整合此技術至AI工廠架構，主打工廠端超快推理的商業場景。

投資觀點

  • 低軌衛星已非概念股：Starlink 800萬用戶規模相當於台灣大哥大或遠傳，已是具體商業服務，而非遙遠夢想；通訊補貼、航空Wi-Fi、國防通訊需求都在快速擴大。
    • 未來有太空履歷（在宇宙中實際運作過）的廠商將成為供應鏈入場券
    • 台灣供應鏈以天線模組和電子零組件為主，量會持續增加
  • LLM模型效能過剩→注意資本支出循環轉折點：若接下來大家對「更強模型」的需求接不住，AI Capex循環可能出現單點突破而非全面提升的局面；觀察Blackwell在2026年中推出後的訓練需求是否真的更旺盛。
  • NVIDIA的Groq整合：代表AI推理（Inference）的速度優化成為下一個競爭焦點；工廠自動化、即時製造判斷等場景最有商業價值，消費者端快速吐字意義不大。

SpaceX / Starlink分析

商業模式與規模

  • SpaceX 2025財務預估：營收約150億美元，2026年預估達220-240億美元；主要來源是Starlink衛星網路服務，而非火箭發射。
  • Starlink規模：800萬用戶、150個國家；對標台灣大型電信商，規模已相當可觀。
  • 商業模式：SpaceX本質上是全球超大型電信商，在基礎設施不足的地區直接搶走原有電信商份額；中東戰爭進一步驗證了衛星通訊的戰略重要性。
  • Google持股：Google累計持有約10%股權；SpaceX上市後Google直接受益。

IPO對市場的影響

  • SpaceX IPO消息帶動低軌衛星全面拉升，包括直接供應商和「沾邊題材」；部分純題材概念股也顯著上漲。
  • OneWeb（英國）、Kuiper（Amazon）等競爭者相關供應商也同步受惠。

低軌衛星供應鏈

太空履歷（Space Heritage）認證

  • 定義：零組件在宇宙中成功運作過，取得「太空履歷」，才有資格進入衛星供應鏈。
  • 重要性：射入宇宙後面臨高輻射（大氣層外無屏蔽）、真空散熱問題（只有輻射散熱，無對流）、極端溫差；有太空履歷代表技術驗證完成。
  • 台灣機會：台灣主要出的是電子零組件和天線模組；天線模組是量最大的需求端（地面端比天上端用量更多）。

現有機會與挑戰

  • 推進器、天線、通訊晶片、CIS感測器等需求都在增加。
  • 台灣新掛牌太空類股良莠不齊：需確認是否有真實的「太空履歷」，或只是廣用零件搭題材便車。

LLM 2025年進化回顧

重大技術突破

  • RLHF延伸方法（RFVR）：新的後訓練強化學習方法讓模型潛力繼續提升；三條Scaling Law路徑（Pre-train、Post-train、Test-time）均On Track。
  • 長文本準確率大躍進：GPT-5.1能回答128K文本中四個隱藏問題的正確率僅47%；GPT-5.2 同等條件下達到97%（256K文本約77%）——Transformer長文本中間遺失問題大幅改善。
  • Gemini Flash成本衝擊：Gemini 3 Flash價格是GPT-5.2的三分之一、速度快三倍、輸出品質大部分場景相近→市場開始依Cost評估模型，而非只求最強。

效能過剩隱憂

  • Sam Altman坦言：大多數人用GPT只是聊天、簡單整理資料，完全用不到「博士等級」能力；但OpenAI還在繼續推進更強模型。
  • 財報狗實際評估：已考慮從GPT-5.2降級到Gemini Flash，主要因為成本差三倍但品質差不多。

主要弱點（Gemini）

  • Gemini 3 Flash的「幻覺率」很高：有答案時準確率高，但沒有答案時會「亂掰」（幻覺輸出）；不適合需要可信賴產出的企業服務場景。

NVIDIA收購Groq技術資源

交易概況

  • 規模：約200億美元（注：非整間公司收購，而是取得Groq技術授權並招攬人才）；Groq今年9月融資估值僅69億美元，此次溢價約3倍。
  • 背景：Groq創始人來自Google TPU團隊，LPU（Language Processing Unit）架構類似TPU，採ASIC設計，專為大型語言模型推理優化。
  • 核心技術：S-RAM（片上記憶體）取代HBM，犧牲容量換取速度；每秒token吐出量達500+（OpenAI約50，Google約150+）。

NVIDIA整合目的

  • 將Groq的LPU整合進NVIDIA AI工廠架構，專攻AI推理（Inference）和即時工作負載。
  • 商業邏輯：工廠端、製造業即時決策需要超快推理，願意付更高價；消費者端速度快「好玩但沒人付錢」。
  • 潛在競爭態勢：類似Google TPU對NVIDIA的壓力，NVIDIA可能有意在推理晶片領域建立更強防線。

Groq的技術限制

  • ASIC架構不相容於CUDA和JAX等主流生態系；需要額外的編譯/轉譯層才能讓既有模型運行在LPU上。
  • NVIDIA介入後，CUDA整合機率大增，原有軟體壁壘有望解決。

2026年LLM發展展望

  • 單點突破時代：不再是每一代模型在所有面向全面碾壓；2026年將看到「在某些具體場景上的突破」，而非整體性躍升。
  • 速度優化成主軸：Groq整合、NVIDIA Blackwell推理端優化，速度將是下一波差異化競爭要點。
  • 企業AI應用標準化：台灣金管會可能發布金融業AI使用guide/認證；各行業垂直標準化是企業AI真正落地的前提。
  • Blackwell 2026年中才能量產：在此之前，「因為算力大幅提升→模型再次全面變強」的劇本不會出現。

節目金句

  • 「Starlink 800萬用戶，不是全藍就都用衛星通訊，至少用衛星上網的人蠻多的——而且這個數字已經相當於遠傳或台灣大哥大了。」
  • 「太空履歷就是你的東西射到宇宙，成功運作一段時間沒有壞掉，那你就取得了太空履歷——代表你的東西已驗證在太空中可以使用。」
  • 「我昨天還在看我們網站要不要改用Gemini Flash，不要用GPT-5.2，因為Gemini Flash價格只有三分之一、速度快三倍，大部分場景品質差不多。」
  • 「我們已經開始比成本了——這件事情代表LLM效能已經快要到那個不再是每一方面都全方位碾壓的時代了。」
  • 「Groq一秒吐出500個token；OpenAI是50個——那種速度快到你閱讀都跟不上，在工廠端、即時決策場景才真的有商業意義。」
  • 「我覺得對這些廠商是一個可以觀察的方向——什麼時候弄的東西越來越準確，然後企業AI真的落地，這個疊代會左右大家的想法。」
  • 「希望這個浪潮可以持續啊——不管是作為使用者給我們更多驚喜，或者是作為投資人給我們更多錢，都要靠你了，AI產業的各位。」
