投資觀點

  • 空杯心態做交易：不要一直抱著既有印象，被動元件稼動率低不代表沒機會；記憶體前車之鑑：稼動率開滿後還是大爆射，需求可能比想像大。

  • 看公司真金白銀的動作，不看小作文：大廠買地擴廠是最直接的信號，客戶一直下達擴產要求才是原因；技術面表態有時也是提前掌握資訊的資金在動。
  • 現在行情與2021不同：2021年是全面牛市，現在是窄行情；指數高位有通膨和企業獲利作支撐，合理性存在，但不代表要追。

CoWoP（Chip on Wafer on Panel）題材

  • 起因：NVIDIA Vera Rubin 後段良率遇到問題，市場認為有機會引入 CoWoP 做替代方案。
  • 外洩資料：曾有一份標有 NVIDIA 工程師姓名的 PPT 在市場流傳，顯示從傳統封裝改為 CoWoP 的方案，且圖上無 Lid（上蓋）。
  • 業界朋友分析：
    • CoWoP 較可能是 Alternation Solution（備選），不一定成為主流；必須與大型 fab 深度合作，fab 廠商不一定願意承擔良率責任。
    • 無 Lid 設計若落地，會改用 Stiffener + 雙層 TIM 散熱，單價不一定比有 Lid 方案低。
  • 歷史前例：M9 材料謠言、PTFE 材料謠言都曾引發題材，但最終未必採用。
  • 操作建議：可以試試 CoWoP 概念股，但不建議直接做空 Lid/MCL 相關股（因替代方案未必成立）。
