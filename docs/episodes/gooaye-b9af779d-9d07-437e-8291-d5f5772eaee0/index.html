三句話總結

  • CPU 趨勢確立：Agentic AI 時代 CPU 負載大增，Intel、AMD、ARM 成新一波關注焦點，Jensen 亦在最新展會端出獨立 CPU 解決方案。
  • 蘋果搶市佔：市場整體手機出貨下修 20%，蘋果接受記憶體漲價並維持售價，預計市佔逆勢成長，中系手機廠受衝擊可達 -40%。
  • 油價是地緣關鍵指標：中東停火仍有變數，維持油價在 90 幾元以下視為「陳平時期」，破 120-130 則需重新評估風險。

投資觀點

  • 美股定性為死貓跳、觀察季線：NASDAQ 反彈幅度大，直接跳過季線；若後續修正不破季線或破後迅速收復，可能視為短底，但仍不排除二次探底。

    • 油價維持高位 = 繼續保守；油價盤整回落 = 可當陳平期做多
    • Vix 尚未死絕前，不加槓桿

  • 台股多頭格局未變：一日多達 8-9 檔漲停，台股 GDP 加 8% 優於亞洲其他股市，中長期仍偏多看待。

  • Tirot 流強策略：反彈時賣出弱勢標的騰出流動性，不在右上角主動減碼強勢標的；有明顯破線訊號才列入減碼對象。

  • 外資動向：部分觀察到外資準備補回台股部位，等同把籌碼交棒給追高的外資。

CPU 趨勢分析

Agentic AI 時代 CPU 角色轉變

  • GPU vs CPU 比喻：GPU 是球場上砍分的球員，CPU 是負責調派指揮的教練；Agentic AI 大量 in/out、API 呼叫、程式執行，CPU 使用率大幅提升。
  • 實驗室數據佐證：Georgia Tech 論文指出 CPU 占 Agentic 工作流總延遲 50-90%；康乃爾研究顯示傳統 AI CPU 佔比約 10%，Agentic AI 可達 40-50%。
  • NVIDIA 表態：最新展會直接展示 CPU Horror Solution，整個機架配滿 CPU；Meta 為使用者之一，確立趨勢。

主要受惠標的

  • AMD、Intel：X86 雙雄同步受惠，AMD 市佔持續吃進，Intel 在 NVIDIA DGX NVL8 的 Host CPU 上有舞台。
  • ARM：伺服器市佔從 2018 年低於 5% 成長至 2025 年 20-25%，在 CSP 超大規模佔比達 50%；AWS Graviton、Google Axion、Azure Cobalt 均為採用案例。
  • NVIDIA Vera CPU：搭載 88 顆 ARM 架構核心，已獲客戶採用，本身亦屬 CPU 類股。
  • 台股純度難識別：多數台廠 CPU/ASIC/GPU 客戶混雜，建議從美股純度高標的為輔助判斷。

Intel × Tesla TerraFab 合作

  • Samsung、Intel 協助 Tesla TerraFab：市場解讀為 Intel 與 Samsung 的利多，對 Tesla 屬中性。
  • Intel 受益邏輯：可收取高毛利授權或顧問費，若 TerraFab 遇阻也可能轉訂 Intel 產能（E-MIP / E-MIP-T 封裝）。
  • Tesla 隱憂依然存在：自行投入製造良率、前段光照良率風險；短期產品預估使用成熟製程（7nm 等級）而非最先進節點。
  • 類比軍火庫概念：如同 NVIDIA 各家 IC Design 都逃不掉，TerraFab 相關廠商中最可能成為「軍火庫」的角色最值得關注。

手機市場結構

  • 全年手機出貨下修約 20%：蘋果、三星維持水位，中系手機廠可能被迫承擔 -40% 的出貨衝擊。
  • 蘋果策略：接受記憶體漲價並擴大滲透，犧牲部分毛利換取市佔，預計全年優於同業。
  • 消費性元件短線危險：供應鏈砍單潮逐漸出現；但 1-2 年後若需求回升，逢低佈局可期。

被動元件與記憶體

  • 鋁電容、鉭電容持續缺貨：AI 需求帶動，有機會成為下一波族群，觀察是否集體轉強作為進場訊號。
  • 記憶體依然缺貨：雖股價進入休整，產業層面供需仍緊。

QA

如何判斷一支股票「在吵什麼故事」

  • 情境：聽眾問法說會以外如何解讀市場當下的股票敘事。
  • 重點回答：以「中探針」為例，A 派認為是 Pogo Pin、B 派認為是 Kyber Rack 用的 Connector Pin，兩邊敘事完全不同但股票都在漲——「一個上漲各自表述」；最終回歸技術面供需，研究故事的目的是預判哪類資金會進場，而非追求精準。

槓桿使用時機

  • 情境：聽眾問如何結合大盤判斷是否開槓。
  • 重點回答：絕大多數情況只在「已賺錢的部位」加槓桿；波動率放大（VIX 高）時主動降槓，避免心性崩潰或被強制斷頭；油價與波動率為輔助指標，非絕對標準。

投資故事重要性

  • 情境：聽眾問基本面故事怎麼找、怎麼判斷誰的故事對。
  • 重點回答：故事重要但非絕對，技術面強勢（有人買）才是核心；以 Marvell 為例，提早佈局是因知道新聞出來後資金會進場，屬於「凱因斯選美理論」的應用。
