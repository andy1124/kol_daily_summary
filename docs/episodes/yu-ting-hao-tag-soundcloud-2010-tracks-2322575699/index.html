提及公司

  • 美股：NVIDIA（輝達）、Boeing（波音）

三句話總結

  • 美債利率破5%：美國30年期公債殖利率突破5%、10年期突破降息循環下降趨勢線，實質利率快速墊高疊加通膨預期，股債雙殺風險浮現。
  • 通膨全面外溢：PPI達6%、CPI近4%，小麥產量創1972年以來最低、番茄價格年增25%創歷史新高，食品通膨開端已至。
  • 川習會成果有限：雙方成立中美貿易與投資委員會，波音拿到200架訂單，但無重大關稅新協議，伊朗問題懸而未決。

投資觀點

  • AI資本支出循環仍確認向上：通膨與利率壓力可能造成股市回調，但AI資本支出不受影響，二、三季財報將比一季更亮麗，回調方向仍在右上角45度。
    • 費半脫離200日均線幅度達62%，歷史上泡沫高點平均僅35%，具估值修正壓力
    • 歷史案例（1989日本、1999年網路泡沫、2007中國泡沫）均在股市仍創高、30年期殖利率突破時出現大幅修正
  • 左側投資機會浮現：台股回調至4萬點，為一至兩個月來首次出現較好的加碼時機，建議多做關注。
  • 市場轉折框架：若要讓股市顯著反彈，需油價回跌、通膨預期降溫、聯準會利率定價轉鴿，三者缺一不可。

美債市場分析

  • 30年期公債殖利率突破5%：最新拍賣得標利率5.046%，遠高於4月4.87%，需求顯著萎縮，為2007年以來首見。
    • 2007年8月量化基金崩盤起點後演變為全球金融海嘯，歷史警訊值得注意
  • 10年期殖利率衝高至4.55%：創一年新高，同步突破2023至2024年降息循環的下降趨勢線。
  • 利率定價結構：約40%漲幅來自通膨預期，更多來自實質利率上升，市場要求更高真實回報；聯準會終端利率預期從不到3%上調至3.85%。

通膨多維壓力

  • 食品鏈全面升溫：美國小麥產量降至1972年以來最低，硬紅東麥乾旱創1957年紀錄；番茄受關稅衝擊（7%進口關稅）及冷鏈成本上升，5月正式調漲。
  • 能源傳導：汽油庫存低於季節中位數約5%，夏季開車旺季將至；台灣等東亞國家政府補貼導致民眾感受低於實際現貨漲幅。
  • 需求型通膨：銅價突破每噸1.4萬美元、年增13%，AI、資料中心、電網升級帶動需求端持續拉升。
  • 農產品普漲：黃豆油、小麥、生質燃料今年平均漲幅約17%。

川習會分析

  • 表面成果：成立中美貿易委員會與投資委員會；波音獲200架訂單；部分農產品採購；約300億美元非敏感商品關稅互減；批准約10家中企採購H200晶片；花旗取得外商獨資券商牌照。
  • 核心問題未解：無正式聯合聲明；無大型貿易協議；晶片管制未全面放鬆；伊朗問題雙方均無突破；台灣問題中方私下強調為中美關係最重要議題。
  • 競爭新格局：「鬥而不破」，中美進入可控競爭階段，雙方在科技、外交各領域全面升級，但確保不引起直接軍事衝突。

美歐裂痕與軍事撤離

  • 大規模撤軍：宣布撤離德國5000名駐軍（德國共3.5萬美軍，為整體歐洲兵力近一半）；波蘭4000人部署喊停；戰斧巡弋飛彈、暗鷹高超音速武器系統同步撤離。
  • 歐洲防衛缺口：缺乏1000至3000公里的深度精準打擊能力，難以在短期內獨立取代美軍威懾力。
  • 美歐價值觀裂痕：川普派系認為歐洲走向高稅收、高移民、綠色威權；歐洲認為美國右派破壞全球秩序，二戰以來最嚴重分歧。
