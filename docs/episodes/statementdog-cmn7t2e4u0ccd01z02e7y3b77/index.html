512.【財經時事放大鏡】開牙科 x 跑外送

財報狗 Podcast｜2026-03-26

提及公司

  • 美股：Grab、Google
  • 台股：南亞科
  • 其他：SanDisk（Western Digital）、鎧俠（Kioxia）、Solidigm、Cisco、Uber Eats、foodpanda

三句話總結

  • 南亞科私募：南亞科引入SanDisk、鎧俠、Solidigm、Cisco私募，背後邏輯是長約替代，本質是DDR4特殊應用端急需穩定供貨，而非技術上的強強結盟。
  • Google TurboQuant：Google宣稱KV Cache可省六倍記憶體，但現有前沿模型早已使用類似壓縮技術，實際增量節省有限，市場情緒反應過度。
  • Grab入台：Grab收購foodpanda台灣業務，以Super App策略（叫車＋外送＋支付＋金融）切入高所得市場，短期補貼戰可能重啟。

投資觀點

  • 私募背後要看策略意涵，非技術背書：南亞科獲得787億元資金不是重點，三家NAND廠以私募而非長約入股，代表更緊密的利益綁定，優先供貨順序甚至優於合約客戶。

    • 鎧俠/SanDisk選私募，美光選五年長約，取捨在於「綁得多緊」和「付出多少」
    • Cisco入股則是因企業級ESSD/Router需要DDR4低功耗記憶體，自製成本不如外包

  • DDR4供需結構性偏緊：一線廠（三星、美光、SK Hynix）忙著做HBM，無暇認證DDR4新客戶；中國廠在拉DDR5也沒回頭；DDR4特殊應用端只能靠南亞科這類二線廠。

    • 整體記憶體產業都很忙，是這次私募事件的背景脈絡

  • Google TurboQuant情緒>實質影響：論文中的六倍節省是與「完全不做任何壓縮」對比，而非現有模型，實際節省幅度大幅低估；且就算省了，Context Window拉長後需求照樣爆。

南亞科私募深度分析

為什麼是私募而非長約

  • 長約（SPA）鎖五年，私募鎖三年；私募是「股東自己人凹一下」，供貨優先序更高
  • 美光選長約；SanDisk、鎧俠選私募，不同風險承擔偏好
  • 投入資金換算成優先供貨的「回本期」是關鍵評估邏輯

Cisco入局的意義

  • Cisco路由器、Switch等企業網路設備需要DDR4，不想自己擴廠，直接入股保供
  • 印證「企業級ESSD需求用DDR4」的利基市場確實存在且被重視

股價反應

  • 消息出來後開漲停，但10點後已轉弱，市場認為庫存利益已在附近高點
  • 短線漲停是情緒反應，長線要看DDR4 Cycle何時Pick up

Google TurboQuant 技術解析

  • TurboQuant 宣稱 KV Cache 可縮減為原來六分之一
  • 關鍵誤解：六倍是與「不做任何壓縮」比較，現有 Claude、GPT-4 等前沿模型早已使用 KV Cache 量化壓縮，實際額外節省幅度遠小於六倍
  • 就算有節省，Context Window 因此拉長，Token 消耗照樣上升（Jevons Paradox 效應）
  • 類比：Sora 被關，Seedance 等中國平台仍在快速成長，「某技術暫停」≠「需求消失」

Grab 收購 foodpanda 台灣

策略背景

  • foodpanda 台灣原本要賣給 Uber，被公平會否決（壟斷理由）；Grab 進入無壟斷問題
  • Grab 目前市值約 18 億美元 GNV，以 6 億美元收購，折讓明顯
  • Grab EPS 從 2023 年開始轉正，2024Q4 EPS 達 0.04 美元，三倍成長

Super App 潛力

  • 東南亞已整合叫車、外送、支付、保險、金融等服務（類似美團模式）
  • 台灣市場是其首個較高所得市場，具備金流套利潛力（移工匯款、小額跨境）
  • 短期最直接效應：補貼戰可能重啟，外送平台費用有望下降

節目金句

  • 「錢本身不是重點，重點是背後隱含的合作關係的意義。」
  • 「一線廠很忙，我在做HBM，你DDR4是自己的事；南亞科就是一個既有在做DDR4又滿大的一間公司。」
  • 「私募是股東嘛，股東自己人凹一下，合約就是個框架。」
  • 「那個縮減六倍是跟我完全不做任何縮減的時候相比，可是現在所有的模型其實都有在做縮減。」
  • 「就算真的縮減，你的 Context Window 就長一點，到時候又爆了。」
  • 「GNV 18 億，誰管你 GNV 多少，你告訴我最後賺多少錢。」
  • 「補貼戰開始了？那我又得利了，食物又要變便宜，超爽。」
  • 「Grab 如果搞跨境金流這件事情，我覺得有搞頭。」
