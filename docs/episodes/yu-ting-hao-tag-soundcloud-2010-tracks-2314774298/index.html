  • 全民炒股潮，新一代年輕人用ETF槓桿放大報酬：0~19歲開戶年增25%，房屋抵押貸款資金流入股市，券商已開始收水停止新增借貸，內部過熱訊號浮現。
  • 中東油價衝擊+PNI假象：數據好看但背後是恐慌性備貨：美國製造業PMI跳到54、台灣PMI達60（4.5年新高），主因是企業怕漲價提前囤貨，而非終端需求真正爆發；儲蓄率跌至3.6%，消費開始透支未來。

投資觀點

  • 三層市場挫位並存：第一層：油價衝擊供應鏈，股市無明顯反應（資金認為油漲股也漲）；第二層：美國PMI假性擴張（備貨推動非需求推動）；第三層：台灣PMI/出口因AI強勢，但傳統製造業與內需仍弱。
  • 資金行為背離是警訊：對沖基金連三周減碼美股，北美股票配置降到歷史低點，並進行過去10年最大規模去槓桿；指數創高同時機構悄悄退場。
  • EPS成長追不上資本支出：AI四大CSP業者資本支出增速7080%，EPS增速只有1620%；若2627年EPS緩步回到1112%，目前估值壓力難以消化。
  • 房市降溫=股市上行：Q1推案量跌至歷史新低（248件，年減約50%），房市釋出資金大量流入股市；央行第七波信用管制讓不動產貸款年增幅從9%壓到3.74%，但房屋抵押貸款則流向股市。
  • 貧富差距惡化是台美共同的中長期風險：台灣基尼係數0.6（高於日本0.54、南韓0.57）；美國前10%握68%財富，後50%只分到2.5%；消費透支儲蓄、民調支持度跌到37%是川普政策的壓力來源。

主題分析

台股突破4萬點

  • 歷史紀錄：單日盤中漲幅1828點（史上最大盤中漲點）、收盤漲幅1778點（史上最大收盤漲點）、首次收盤站上40,000點（收41,031點）。
  • 市值排名重洗牌：台積電（第1）→台達電（第2）→聯發科（第3）→鴻海（第4）→日月光（第5）。
  • 外資仍未回補：三大法人合計買超780億，但主力是內資個股買盤；外資空頭格局未變，行情由內資主導。
  • 六大投顧目標：台新4.4萬（5月）、統一最高預估6萬、富邦4.1萬4.5萬（全年）、兆豐5萬，多數預估26年高點在46萬。

全民炒股：新一代投資人結構

  • 開戶人數1420萬：普及率61%，單月增加10萬人；0~19歲開戶年增25%，成長最快。
  • 資金來源多元：ETF資金、銀行閒置資金、房市釋出資金、房屋抵押貸款，全部湧進股市。
  • 槓桿模式：用房屋抵押貸款（低利率）→買ETF→再質押→再借→循環槓桿；多家券商已停止新增借貸、要求到期實質還本。
  • 行為特徵：相信大盤長期創高→買ETF或0050→再放大槓桿超越ETF報酬，自認「保守」但實際上已高槓桿。

中東油價衝擊

  • 霍爾木茲海峽近乎封鎖：美軍護航計劃啟動，但航運業者對護航範圍存疑；目前通過量從最高25艘/天歸零。
  • 庫存消耗加速：OECD庫存僅剩32天；夏天可能逼近操作下限。
  • 伊朗策略：用時間換空間、油輪浮動儲油（已存數千萬桶）、最低限度產出防止停擺，預計5月底才面臨儲存極限。
  • 油價 vs OVX背離：油價衝到110美元以上，但OVX（油價波動率）回落，市場把高油價視為長期結構性問題而非短期恐慌。
  • 委內瑞拉鬆綁：美國對委內瑞拉制裁鬆綁，出口恢復至100萬桶/天，加上俄羅斯制裁鬆綁，正在補足全球供給缺口。

PMI假性擴張與經濟隱憂

  • 美國製造業PMI 54：突然從52跳到54，主因是企業提前備貨（怕漲價），而非終端需求增長。
  • 台灣PMI達60：4.5年新高；背後是記憶體報價上漲備貨、第二季樂觀預期提前布局、電子零組件持續調漲。
  • 美國GDP Q1年化2%：低於預期2.3%；消費1.6%、投資全部來自AI基建（非住宅固定投資AI設備7.85%），75%成長來自AI。
  • 儲蓄率跌至3.6%：2021年高點8%→現在3.6%，接近疫情低點；薪資成長低於消費成長，美國人開始透支未來。

台灣貧富差距與財政結構

  • 基尼係數0.6：高於日本（0.54）、南韓（0.57）；AI行情讓科技業高薪族快速累積財富，傳統製造業/服務業薪資停滯。
  • 稅收結構偏向直接稅（70.8%）：公司稅占30%已超過個人所得稅（25%），高度綁定AI景氣；AI好稅收大增，AI不好稅收大減。
  • 傳統製造業仍在緩步復甦：景氣燈號在綠~黃藍燈游走，4月新車掛牌年減4.5%，內需消費動能仍弱。
