三句話總結

  • AI Insurance（Lemonade）：Lemonade 透過 Tesla API 串接駕駛行為數據，實現個人化定價保費，從「群體統計」進化為「即時行為定價」，代表 AI 在金融保險業的破壞式創新模式。
  • AI 基礎建設仍未演完：Micron 財報顯示 DRAM 及 NAND Flash 全線吃緊，記憶體漲價動能持續；美股光通訊（Lumentum、Coherent）與台股散熱族群同步走強，Infrastructure → Platform → Software 輪動尚未完成。
  • 衛星題材升溫：美股（Cyber Systems）與台股衛星概念股同步出現資金輪動跡象，SpaceX Starlink 持續擴張 + 美國政府表態維持太空領先，IPO 前仍是布局視窗。

投資觀點

  • Infrastructure 未必已結束：Micron 財報明確顯示 AI Server DRAM（包含 HBM）、NAND 全線供不應求；消費性記憶體 2026 上半年乃至全年可能衰退，整體均價仍強。
  • 光通訊持續強勢：Lumentum 與 Coherent（Cockeron）在美股持續維持強勢；台股散熱族群年底再度摸前高，顯示 Infrastructure 資金輪動尚未轉向。
  • 動能 + 長抱雙策略：動能題材（衛星、散熱、記憶體）以試單插旗後邊打邊加；小規股（小型台股）長抱不動，以持有理由消失作為出場條件。
  • Transition 期部位管理：目前手上多樣標的是正常現象，等待新主流明確走出後，才把弱勢標的拆掉集中加碼。

Lemonade：AI 保險模式分析

  • Tesla API 串接：不需硬體 OBD 感測器，直接串接 Tesla 車載數據，獲取駕駛習慣（急煞、車距、夜間行駛、使用 AP 與否）精算個人化保費。
    • 夜間使用 Autopilot → 保費不因夜間行駛加費；手動夜間高速行駛 → 保費較高。
  • Bundle 商業模式：車險獲客後 upsell 家庭產險、旅遊保險，提升 ARPU。
  • 傳統保險公司轉型阻力：內部人員利益衝突、舊 IT 架構資料格式碎片化（找不到改舊軟體的人）、決策層保守觀望，給新創彎道超車機會。
  • Lemonade 風險：歷史賠付率偏高，現金流轉正後敘事才成立；仍需觀察其大規模商業化後的賠付控制能力。
