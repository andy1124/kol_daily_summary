AI 重點摘要

提及公司

  • 美股：NVIDIA（輝達）、Amazon（AWS）、Google、Broadcom（博通）、Flex、Credo（Cradle）、Kintech（Kinsus / 景碩）、AirLab
  • 台股：台積電（TSM）、貿聯（Bizlink / 3665）、欣興（Youngtek）

三句話總結

  • AWS 活動後供應鏈反應錯方向：AWS reinvent 宣布採用 NVLink Fusion 屬於雙軌設計（NVLink + 本家 ASIC 並行），主委認為市場錯殺 AWS 供應鏈；真正受傷的應是 Broadcom SUV 相關陣營，而非 AEC/NVIDIA。
  • 散熱降價壓力浮現：Coldplate 零組件遭遇殺價（市場傳六折，確認約七至八折），QD 廠商也因 supplier 達 50 家而面臨降價；但市場仍在擴張，Q 量上升有望 offset P 下滑。
  • 散裝航運 BCI 值得追蹤：BCI 指數大幅上漲，背後故事是烏克蘭停戰預期加上西非新鐵礦產能，主委詳解 Capesize 現貨船 vs. 長約船的選股邏輯，建議等事件確認後不怕高進場。

投資觀點

  • 動能轉爛，轉向逆向佈局：當前盤面追漲很容易「不勝被套」，主委轉為「等東西掉下來再撿」策略——看好趨勢的標的，跌下來就補倉，不追高位動能股。
  • 指數部位增加：動能倉位止利後轉入指數，等待下次破季線後大幅加碼；維持「大指數 + 前瞻小市值」的啞鈴配置。
  • AWS 供應鏈被過度悲觀定價：NVLink Fusion 是整合而非取代，真正的基本面沒有改變，「大家都已經共同期待的事件」兌現後往往反而上不去，需要等情緒消化後再進場。

AWS reinvent — NVLink Fusion 解讀

  • 雙軌設計：AWS 的 NVLink Fusion 是讓 NVIDIA GPU 與 AWS 自家 XPU（Trainium/Inferentia 等 ASIC）整合，並非放棄自家 ASIC 全押 NV。
  • PCIe 路線並存：AWS 仍維持本家 PCIe 架構路線，NVLink Fusion 是額外一條可選路徑。
  • 真正的受害者：Broadcom 相關的 NVLink 聯盟（新宣布的 AI 晶片合作陣營），在這一波中優勢未顯；AEC、NVIDIA 本身、AWS ASIC 供應鏈反而是受惠方。
  • 市場錯誤解讀：大量 AWS 供應鏈標的跳水，主委認為是「情緒反應錯方向」，不改中長期趨勢。

散熱供應鏈壓力

  • Coldplate：Group A 零組件遭殺價，市場傳六折，實際確認約七至八折。
  • QD（Quick-Disconnect）：supplier 超過 50 家，降價壓力確定。
  • 影響評估：若伺服器出貨量繼續放大（如 Google Chain 有 upside），P 下滑可被 Q 增加抵消，整體仍可維持；但若出貨量不如預期，估值壓力較大。
  • Kinsus 相關：NV 未要求其在 NV PCIe 相關專利上讓步，僅在價格上稍做協商，仍有望進入下一代 RVL（Rubin）。

散裝航運 — BCI 觀察

選股邏輯

  • 挑現貨船：BCI 漲價直接反映在現貨報價上，長約船按固定租金收費不受惠，需挑現貨船比例高的公司（美國 SBLK、台灣玉明）。
  • Capesize（海峽型）：無法通過運河、需繞行的最大型散裝船，是 BCI 指數的主角。

背後故事

  • 烏克蘭停戰預期：市場資金在賭停戰後戰後重建的大宗物資運輸需求，有船隻提前停靠港口等待。
  • 西非新鐵礦產能開出：基本面因素，但影響程度小於停戰故事。

操作策略

  • 現階段屬於「賭注」性質，非確定性基本面。
  • 等事件（停戰）正式發生後再不怕高進場，勝率更高（類似記憶體確認缺貨後追進邏輯）。

Cradle（Credo）財報觀察

  • Cradle 財報大幅 beat 預期，先跳漲 23%，後收斂至 +10%，確認了基本面正確性。
  • 台灣貿聯（Bizlink）作為 Cradle 客戶，財報開出後直接高開低走，技術猛男藉此判斷 Cradle 也會出現相同走法並在最高點賣出。
  • 教訓：太多人共識同一件事（Cradle 一定 beat），反而導致好消息兌現後「上不去」。

EMIB 設備股選股

  • 多數被市場貼上「EMIB 概念股」的台灣設備廠，實際上與 Intel 關係不佳或無供應關係，需細拆後才能找到真正受惠標的。
  • EMIB 時程仍長（Google TPU V9 約 2028–2029），題材性大於基本面確認；適合等確認後不怕高進場，或題材炒完後反手做空。
