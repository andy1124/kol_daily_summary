投資觀點

  • 不要用股價短期表現驗證基本面：關稅利多公布後台股上漲就說是好事、隔天跌就說是壞事，這種邏輯很危險，基本面要單獨判斷。
  • 設備廠物最確定的方向：台積電告訴你 CAPEX 數字就是要拉，接下來三年看往100B的方向去，這不是題材，是確定的數字與邏輯。
    • 台灣 frontrunner 先拉一波，美股設備股繼續漲，說明台灣的賣壓只是獲利了結，後面等均線整理完畢再看加碼時機。
  • 被動元件行情難掌握：日系廠轉向高規格後釋出中低階訂單給台系的邏輯類似記憶體排擠效應；但稼動率低的品項不代表沒機會——記憶體的前車之鑑：稼動率開滿後還是大爆射。
    • 電容 > 電感 > 電阻（電阻稼動率偏低，漲價訊號較意外）
    • TLVR 電感可能是後面最缺的品項
  • 現貨報價陷阱：記憶體現貨市場相對小，聰明資金早幾十分鐘就知道，散戶看到11點才去追是給人家出貨；真正重要的是合約報價，模組廠持續真金白銀買顆粒才是最可信的信號。

台灣關稅談判

  • 對等關稅15%，與日本、韓國、歐盟持平。
  • 已簽署 MOU，取得232條款最惠國待遇——未來任何行業被調查若談出優惠，台灣優先適用。
  • 重要背景：以前台灣因特殊政治地位無法加入自由貿易協定，競爭上吃虧；現在川普大刀一揮後，大家都被拉上檯面，台灣反而站到與主要競爭對手公平競爭的位置。
  • 台積電等民間企業負擔美方2500億投資的主要部分（非政府出資），是企業自行赴美擴產，客戶需求所致。

台積電法說會重點

  • CAPEX 52-56B：市場預期45-48B，主持人預期50B，實際 Surprise，"52到56，Surprise Motherfucker"。
    • 宣示接下來三年朝100B方向邁進。
  • 8吋產能釋出：台積電把8吋產能飛燒，轉給世界先進（Vanguard）——台積電的「兒子」最大受益，部分聯聯也有機會。
  • N2 量產進度快：N2 良率超預期，N2 相關特化股前段已先拉波，後續均線回測後可能是加碼機會。
  • 設備廠物後市確定：設備廠物2026-2028年基本面非常強勁；股價不一定，但營收獲利確定會很好。

被動元件觀察

  • 豐華高科（中國）帶頭漲價 → 國巨等台系跟進漲價，電容電感壓低均已漲。
  • 日系廠商因 AI Server 用到更高規格被動元件，把中低階訂單釋出給台系（邏輯類似記憶體大廠轉向 HBM）。
  • 業界共識：電容最缺 → 電感次之 → 電阻排最後。
  • 電感 TLVR 可能是後面非常缺的品項。
  • 連電阻都開始漲價（稼動率原本不高，意外訊號，但不一定形成連續漲價）。

操作策略

台積電法說後的設備廠物

  • 台積電法說前先持有設備廠物，預計法說後數字超預期就加碼。
  • 法說後台股開高走低（frontrunner 出貨），美股設備股持續漲 → 台股只是清洗賭題材的人。
  • 策略：等均線持續上揚、法說高點被突破時視為下一個加碼點。
  • 保守者：等開最高位置的套牢人解套後再進（套牢者解套通常會賣掉，賣完後車子正式啟動）。
