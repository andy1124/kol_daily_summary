投資觀點

  • 市場集中度風險需留意：S&P 500目前僅約12%成分股在創歷史新高，但指數已創高；同時有5%成分股正在創52週新低，此格局過去僅在1929、1973、1999年出現。
    • 前十大成分股佔指數40%市值，類似巨頭行情；通膨已傷害部分傳產股，但尚未傷及AI
  • 台股ETF化帶來新型態買盤：主動型ETF年規模接近7000億台幣、挑戰兆元，形成類法人穩定買盤，「除非你特意賣，否則股價很難跌」。
  • 台灣在美中博弈中的結構性受益：美中關稅互打期間台灣轉單受益；無人機大疆遭禁後台灣可補位全球市場缺口；成熟製程限制打壓華紅，世界先進等台廠受惠。

川習會成果解析

  • 正面成果：波音200架訂單（重新打開中國市場）；花旗獲外商獨資券商牌照；H200晶片批准約10家中企購買；約300億美元非敏感商品關稅互減；9月習近平訪問白宮確定。
  • 未解議題：伊朗問題無重大突破；台灣議題中方私下強調仍是中美關係核心；晶片管制未全面放鬆；無正式聯合聲明。
  • 結構性意義：中美從全面對抗轉向「逗而不破」的可控競爭；9月訪問白宮前雙方難有大衝突。

美中科技戰動態

  • 美國對中制裁升級：五角大廈「中國軍方企業」名單6年增至187家（2020年僅20家），涵蓋AI、半導體、通訊、無人機、量子運算、雲端基礎設施。
  • 華紅半導體受限：美商務部要求Lam Research、KLA等美系設備廠暫停向華紅供應設備與材料，成熟製程管制範圍擴大。
  • AI軍事化全面加速：美國2027年軍事預算1.5兆美元（年增4.37%），為二戰以來最大擴張，佔聯邦預算比例創40年新高；中國飛彈相關上市企業增至81家，涉及航天科工、航天科技兩大央企，40%創歷史最佳營收。

無人機產業格局

  • 中國壟斷現狀被打破：大疆佔全球市場50%以上；美國禁用大疆後，台灣、韓國、日本、土耳其可補位全球需求缺口。
  • 烏克蘭合作：美烏正式簽署協議，允許烏克蘭軍用無人機出口美國進行測試，未來可能成為美軍採購品項。
  • 反制壓力：北京已宣布全鏈條管制無人機（飛行、生產、銷售、運輸），恐衝擊民用市場。

  • 「市場預設兩件事情一定會發生：第一，荷姆茲海峽再怎麼嚴重它都不會變成停滯新通膨；第二，AI的投資熱潮會持續轉化成企業的獲利成長。」
