股癌 EP638 🐈‍⬛｜2026-02-21

提及公司

  • 美股：Tesla、Anthropic（Claude）、Cloudflare、Palo Alto、CrowdStrike、Confluent、Perplexity
  • 台股：聯發科、台積電

三句話總結

  • 農曆新年後台股展望：槓桿資金與外資在長假前撤出，假期結束後約有 1-2 週的補回買盤，但美股較弱需提高警覺。
  • 美國關稅 IEEPA 被最高法院擋下是偽利多：主持人認為這是負面事件，川普將援引更多法條繼續加稅，並可能引發更多不確定性。
  • AI 取代軟體論是市場過度反應：Anthropic 推出 Cloud Security 功能後資安股全跌，但主持人認為這誤判了 AI 與資安業務的實際競爭關係，軟體 API 化趨勢中仍有贏家。

投資觀點

  • 警戒態勢不變：從年前的偏謹慎到現在，主持人仍維持警戒，原因包括：伊朗戰爭風險升溫、美國關稅不確定性增加、亞洲股市卻強到異常（看不懂的盤更危險）。

    • 因看不懂，做法是持續打洞（多一點現金緩衝），不與市場對做

  • 美股持倉策略：假期前已減碼台股較多，美股為長期持有故少動；若有想買的東西買慢一點，有想賣的稍微多賣一點，讓心理適應各種情境，避免恐慌。

  • 軟體股的 API 化機遇：AI 時代 User Interface 重要性降低，軟體走向 API 化；擁有 AI 無法取代的「非公開專屬資料」的 SaaS 公司，護城河最堅固。

美國關稅情勢分析

  • IEEPA 被最高法院擋下：主持人認為這是利空而非利多，市場反彈是「跌很久後的超賣回補」，不是真正的利多解釋。
  • 川普後續手段多：IEEPA 被擋，但川普手上還有 Section 232、Section 301、Section 122（Universal 10% 關稅）等法源，會換方式繼續加稅。
  • 不確定性增加：對加拿大、歐盟等本就對抗立場的方，可能藉機發揮；川普反擊 → 對方反擊 → 情勢持續升溫，市場最不喜歡不確定性。
  • 關稅退回可能性極低：退關稅等於否定自己，政治後果太大，主持人有 98-99% 信心關稅不會退。

AI 與資安股過度反應

  • Anthropic Cloud Security 的實際定位：這是 SAST（靜態程式碼掃描）工具，幫助找出 Vibe Coding 產生的漏洞，與 Palo Alto、CrowdStrike 等資安業務方向不同。

    • CrowdStrike：針對即時動態威脅攔截，本質上不同
    • Cloudflare：主要擋暴力流量、API 取用保護，也不衝突

  • 市場過度解讀「黑白思維」：看到 AI 能做一件事就直接認定整個產業死亡；但資安 TAM 因 AI 使用普及反而擴大，AI 生成程式碼越多，安全漏洞越多，資安需求越大。

軟體股 API 化趨勢

UI 重要性降低，Application 層仍關鍵

  • API 化時代來臨：機器人/AI Agent 直接取用軟體 API，User Interface 不再是競爭關鍵；能提供最好 Result 的 API 成為最有議價權的東西。
  • 軟體不會消失：老黃（Jensen Huang）也明確反對「軟體已死」論；重新發明輪子的成本遠大於直接 API 串接既有軟體。
  • 護城河關鍵：非公開資料：AI 爬不到、只有這家才有的資料庫，是最難被取代的 SaaS 護城河。
