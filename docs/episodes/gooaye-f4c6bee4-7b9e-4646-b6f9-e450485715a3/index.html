  • CPU 成 Agentic AI 新主角：NVIDIA 展會直接端出 CPU 解決方案，AMD/Intel/ARM 受益明確；CPU 在推論工作流中的負載可達 50% 以上，缺貨信號初現。
  • 鋰（旅）供應受中東衝突衝擊：兩座煉廠被炸，鋰相關被動元件（鋁電容、鉭電容）缺貨加劇，但 MLCC 長期替代趨勢將限制漲價空間。
  • 塑化骨有條件受惠：若美國取得中東關鍵勝利、切斷伊朗廉價供油給中國，台灣塑化廠競爭力回升是可能情境。

投資觀點

  • 盤面難做、維持多頭但降低槓桿：美股波動大，大部分科技股持續走弱；目前把注意力放在相對強勢標的（光通、CPU 類股）。

  • 不要在右上角主動減碼：只有當強勢標的明確破線或未來幾天走弱才列為減碼對象；目前台股還是強於美股。

  • Tirot 流強為核心操作：台股標的眾多，大家同步飆漲，不用刻意找特定族群；美股相對容易識別強弱，以美股為輔助判斷資金輪動方向。

  • 被動元件值得持續追蹤：幾年一次的被動元件大循環有機會啟動，觀察是否集體轉強作為確認訊號。

CPU 趨勢分析（Agentic AI 時代）

為何 CPU 重要性上升

  • 傳統 AI 訓練：GPU 史丹利效率可達 80-90%，CPU 僅是配角。
  • Agentic AI 推論：大量 Back-and-forth、API 呼叫、資料庫查詢、程式執行；CPU 負載可達 40-50% 以上，成為瓶頸。
  • 三家執行長背書：
    • AMD CEO Lisa Su：「Agentic 工作流中執行的大量任務，都是傳統 CPU 任務。」
    • Intel CFO：「Misjudged the pace of demand，CM6 目前產能 Absolutely Constrained。」
    • NVIDIA Jensen：「未來會有更多獨立 CPU 部署。」

主要受惠方向

  • AMD、Intel：X86 兩強同步成長，AMD 市佔持續擴張但不代表 Intel 不增長；Intel 在 NVIDIA DGX NVL8 的 Host CPU 有舞台。
  • ARM：伺服器市佔 2018 年不到 5%，2025 年達 20-25%，CSP 超大規模佔比 50%。
  • CSP 自研 CPU：Google Axion、AWS Graviton、Azure Cobalt——大量採用 ARM 架構。
  • NVIDIA Vera CPU：內建 88 顆 ARM 核心，已獲客戶採用。
  • 台股識別難度高：台廠多混合 CPU/ASIC/GPU 客戶，純度不易分離；建議先從美股鎖定純度高標的。

鋰（Lithium）與被動元件

供需緊張

  • 中東兩座煉廠被炸：鋰（旅）供應受衝擊，旅假（鋰鹽）開始逼近波段新高；雖兩廠不在 AI 供應鏈關鍵位置，情緒仍可推升整體鋰價。
  • 傳導鏈：鋰價↑ → 鋰電池相關被動元件漲價（鋁電容、鉭電容）→ 鋰博（鋰相關模組）漲價。
  • 鋁電容、鉭電容確認缺貨：AI 時代大量需求，加上戰爭衝擊，短期仍維持缺貨狀態。

MLCC 長期替代風險

  • Samsung 電機等廠商方向：MLCC + TLVR 電感組合有望取代部分鋁電容/鉭電容功能，壓縮長期漲價空間與估值倍數。
  • 操作含義：鋰相關被動元件題材屬「事件題材」，估值上限可能不如光通；適合作為對沖中東戰爭風險的配置。

塑化股受惠情境

  • 邏輯：伊朗廉價石化原料供應中國，導致台灣塑化廠長期被壓制；若美國取得中東戰略勝利、鎖住伊朗供貨，台灣塑化廠競爭力回升。
  • 潛在風險：①油價回落、戰爭迅速結束；②MLCC 等替代材料逐步普及，壓縮長期成長空間。
  • 適合作為：戰爭持續加劇情境下的避險配置，非核心多頭部位。

QA

選股方向：題材型 vs 基本面型 vs 動能型

  • 情境：聽眾持有電子紙（8069）虧損逾 60%，詢問反省方向。
  • 重點回答：先釐清自己「為什麼買」——題材型買進者目前找不到任何催化劑；基本面買進者若仍看好，越跌越應加碼；動能型不應在破底時才考慮停損。沒有人知道你的買進理由，只有自己才能找到答案。

空方操作思維

  • 情境：聽眾反映做多抱不住，做空反而較自在，問是否應順著心性做空。
  • 重點回答：做多期望值長期優於做空（上漲無限，做空最多 100%）；若績效告訴你答案，不如改善「抱不住」的問題，而非選擇做空；建議記錄交易，避免感覺騙人。

疑似辨識錯誤

  • 「旅假」→ 建議：「鋰鹽」（逐字稿通篇以「旅」代替「鋰」，係發音相近的辨識錯誤）

  • 「談電」→ 建議：「鉭電容」（「談」為「鉭」的誤識，tǎ/tán 發音接近）
