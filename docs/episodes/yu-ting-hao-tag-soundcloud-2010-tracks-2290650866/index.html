  • 底部判斷需三條件同時成立：伊朗公開承認條件、實質停火進展、荷姆茲海峽恢復正常通行，缺一仍屬短期情緒反彈，非趨勢反轉訊號。

中東局勢與油價

停火進展

  • 川普喊話突破：釋出「只要伊朗同意永不持有荷姆茲」即可停火；4至6週軍事時程暗示最快3月底至5月初結束。
  • 伊朗部分讓步：官方允許非美籍船隻通行荷姆茲，但未確認與川普達成協議；Polymarket預測5月底停火機率超50%、6月達66%。
  • 軍事部署持續：82空降師已抵達中東，海軍陸戰隊兩個遠征單位到位，邊打邊談雙軌博弈成形。

油價與通膨傳導

  • 年初至今油價漲幅達66%，遠超過去40年任何年份，且在兩週內一次到位，與過往緩漲格局不同。
