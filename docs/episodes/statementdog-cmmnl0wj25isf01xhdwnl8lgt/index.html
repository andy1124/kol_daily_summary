509.【財經時事放大鏡】好貴的油！好貴的？？

財報狗 Podcast｜2026-03-12

提及公司

  • 美股：Meta、Google、Microsoft、Palantir（提及）
  • 台股：台塑、台塑化、中纖（提及）
  • 其他：foodpanda（提及），中國塑化廠（古雷等）

三句話總結

  • 荷姆茲海峽危機推升油價：美伊衝突延燒，伊朗宣布擴大攻擊範圍至中東雲端資料中心，荷姆茲海峽受阻讓油價從70美元短期飆至90幾美元；台灣80%原油來自中東，直接衝擊塑化原料供應。
  • 塑化產業進入停盤/封盤模式：上游無法取得原油、庫存約30天，台塑等相關廠商紛紛停盤不報價；過往颱風、能耗雙控等事件都曾引發類似行情，中上游塑化/特化廠受益，下游纖維廠受害。
  • 石油衍生品無所不在：塑膠、合成橡膠、化學添加劑、清潔劑、聚酯纖維等幾乎所有工業/民生材料都源自石油裂解，油價飆升引發整條供應鏈連動漲價，接近民生強需求者最容易轉嫁。

投資觀點

  • 中上游塑化廠：受益：持有庫存的廠商在油價飆升後，舊料按新價賣，利差（Spread）放大；停盤後開盤漲價，配合強需求客戶轉嫁能力強。

    • 參考歷史事件：2017年美國颶風吹毀德州港口，2021年中國能耗雙控，皆有類似行情

  • 選擇「強需求/難替代」的下游：工業用化學品（每天必須生產用，不能停採購）、食品添加劑、清潔劑等民生必需品，轉嫁能力強；反之，終端消費性纖維/成衣廠轉嫁困難，屬受害方。

  • 特化廠（半導體用化學品）：若下游是半導體客戶（可漲價），加上本身有半導體溢價能力，等於雙重受益；上游油漲→下游半導體照漲→夾在中間的特化廠庫存增值且可抬價。

  • 行情速度快但短：此類事件歷史上行情持續週期不長，需注意利差放大後縮小的速度；適合短期操作，不適合長抱。

美伊衝突與荷姆茲海峽

最新動態

  • 伊朗宣布將攻擊範圍擴大至中東雲端資料中心（微軟、Google、Meta 等）
  • 川普喊快速結束，但伊朗需先累積籌碼才會談判，短期難解
  • 油價在純消息面主導下劇烈波動：70→110（峰值）→90→因戰備除油短暫跌回→再漲

台灣的直接衝擊

  • 台灣原油進口：80% 來自中東、20% 來自美國
  • 80% 中東原油全部受荷姆茲海峽影響
  • 部分船隻無法通過，導致台灣塑化廠油料供應中斷，被迫封盤不報價

石化產業供應鏈解析

石油→塑化的完整路徑

  1. 原油輸入 → 裂解（分餾出乙烯、苯、丙烯等）
  2. 基礎石化：乙烯→PE（聚乙烯）；苯→PTA（純苯二甲酸）
  3. 下游合成：PTA + EG → 聚酯纖維（衣服）；CPL → 尼龍
  4. 終端製品：塑膠粒、清潔劑、合成橡膠（輪胎）、碳黑（電池/輪胎）、化學添加劑（食品）

庫存現況

  • 一般塑化廠庫存約 30 天，液態品體積大難存放
  • 部分可轉為固態（塑膠粒）較易存放，液態品轉存則受限於儲槽容量
  • 台灣主要煉化廠（台塑、台塑化）庫存消耗後，若補料困難則只能降載

利差放大邏輯

  • 上游拿不到油 → 舊庫存變稀缺 → 按新油價售出 → Spread 放大
  • 歷史案例：2017年颶風（美國）、2021年能耗雙控（中國），都曾出現塑化族群急漲
  • 觸發點：停盤→開盤後報價上漲→下游不得不接→再漲

各產業受益 / 受害分析

受益（轉嫁能力強）

  • 中上游石化廠（台塑、台塑化等）：有存貨且可按新價報價
  • 特化廠（電子級/半導體用化學品）：下游半導體客戶有支付能力，雙重受益
  • 工業用化學品廠：客戶為工廠，採購剛性高，難以中斷
  • 清潔劑廠：民生必需，毛利率高（超過50%），小漲無痛

受害（轉嫁困難）

  • 下游纖維廠/成衣廠：消費者可以少買衣服，廠商難以直接漲價轉嫁
  • 大型通路商的上游供應商：大客戶議價能力強，難以轉嫁漲價
  • 純加工型中下游：原料漲、成品難漲，利差被壓縮

補充話題：短劇市場

  • 2025年全球短劇市場 110 億美元，2026年預估 140 億美元
  • 中國佔 80 億，美國市場約 15 億（Real Shorts 等 App）
  • Real Shorts 用戶每天平均花費 35.7 分鐘（高於 Netflix 的 24.8 分鐘）
  • 製作成本結構：9 成行銷費用、1 成製作費用；拍攝一部短劇僅需 4 天
  • 投資觀點：平台商目前仍在燒錢投廣告，Recurring 收益模式不穩定，難以長期投資
