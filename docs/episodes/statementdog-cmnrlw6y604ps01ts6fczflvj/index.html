515.【財經時事放大鏡】Intel & 半導體耗材

2026年4月9日 · 48 分鐘 · ✦ 已摘要
AI 重點摘要

提及公司

  • 美股：Anthropic（Claude）、Meta（Llama / Muse）、Intel、Google（Cowork）、Microsoft（Copilot）、OpenAI、Amazon（Broadcom 合作夥伴）、AWS、Broadcom
  • 台股：宋勝（CMP 研磨墊）、中山（鑽石碟）、慶康、有達
  • 其他：福吉堡（Fujibo，日本，CMP 研磨墊）

三句話總結

  • Anthropic ARR 爆速成長：去年底 ARR 90 億美元，2 月升至 191 億，4 月初公佈達 300 億，兩個月成長超過 50%；AI Inference 需求確認呈指數增長，對 AI 算力資本支出無任何疑慮。
  • Intel 玻璃基板與 EMIB 封裝：Intel Glass Core 技術持續推進，EMIB 橋接封裝成為台積電 CoWoS 產能緊缺下的外溢出口；台灣玻璃基板聯盟（ECO）持續關注，量產時程預估延後至明年或更後期。
  • 半導體 CMP 耗材族群：宋勝推出 Soft Pad 新品，目標打破福吉堡壟斷；CMP 耗材屬量產後需求，順序為廠屋工程→設備安裝→耗材爆量，可視為半導體擴廠週期的後段受益標的。

投資觀點

  • AI Inference Token 需求指數級上升：Cowork（Claude）深度整合工作流程後，企業日耗 AI 費用以萬美元計；Token 消耗量 Exponential 增長已確認，對 AI CapEx 完全無憂，2025-2026 算力採購仍持續。

  • Intel EMIB 外溢訂單邏輯：台積電 CoWoS 產能見頂 → AI 晶片客戶轉向 Intel EMIB 封裝作為備援 → Intel 意外成為先進後段封裝的另一選擇；玻璃基板可與 EMIB 搭配，屬技術排列組合，不互斥。

  • CMP 耗材時序觀察：廠屋工程已啟動，設備安裝跟進中，耗材爆量預計在正式量產後（今年底或明年初）；目前適合研究、觀察進廠進度，尚未到追入時機。

主題分析區塊

Anthropic ARR 成長速度

  • 三個月 ARR 數據：去年底 90 億美元 → 今年 2 月 191 億美元 → 4 月初 300 億美元（配合 Broadcom 合作公布）
  • AR（年化營收）定義：月收入 × 12 的推算數字，非已實現收入；但成長速度仍極具意義
  • 驅動因素：Cowork / Claude API 深度整合工作流，Inference 用量呈指數成長；台灣在全球 Claude 使用量排名第 39（116 個國家中），顯示普及仍有大空間

Intel 玻璃基板與 EMIB 封裝技術

  • EMIB（Embedded Multi-die Interconnect Bridge）：Intel 自有 Bridge 封裝技術，已存在多年；目前被 AI 晶片客戶視為 CoWoS 外溢選項
  • 玻璃基板（Glass Core）優勢：CTE（熱膨脹係數）匹配更好、平整度高、線寬線距更細；難點在於玻璃鑽孔（TGV）製程複雜度極高，良率提升需時
  • 量產時程：日本廠商已延誤，韓國廠商（如 Absolics）仍在推進；台灣玻璃基板聯盟（ECO）持續參與，整體量產預估至少再等 1-2 年
  • ABF 關係：玻璃基板替換的是 Core 層，ABF 增層製程仍保留，非完全替代

半導體 CMP 耗材族群

  • 宋勝新品：推出 Soft Pad（軟研磨墊），從硬墊延伸至軟墊，目標挑戰福吉堡（Fujibo）市占；Q1 2026 送樣認證中
  • 中山（鑽石碟）：為 CMP Pad 磨耗耗材，已從台積電客戶延伸至 Intel
  • 耗材需求時序：廠屋工程 → 設備安裝 → 二次配工程 → 試量產 → 正式量產（耗材需求爆量）；目前尚在前兩段，耗材爆量預計今年底至明年

節目金句

  • 「Entropic 在就是 Cloud 他們在去年年底公布的 AR 是 90 億美金，然後到了今年的 2 月是 191，4 月初公布的是 300 億，欸大家超扯的。」
  • 「自從這種 Cowork 出來以後我完全不懷疑大家的 AI 資本支出會有什麼問題，甚至明年我也不懷疑。」
  • 「台灣在全球 Cloud 的用量是第 39 名，它總共列出來的國家就 116 個而已，台灣的用量在全球其實相對人來說算蠻後面的。」
  • 「你不論在找哪一個（玻璃基板），好像最後也都跟面板廠有關，因為前面沒搞頭啊，只是不知道哪一條路而已。」
  • 「廠物工程先，再看到設備，再看到耗材；你先看到廠屋工程耗材就賺了，那這個大家可以大家想一下觀察月晚發酵的股價也比較晚動。」
  • 「我現在認為說現在錢很難賺，那個分析師在 AI 出來後，統攬，你現在可能要打卡賺這個皮肉錢，好辛苦。」
