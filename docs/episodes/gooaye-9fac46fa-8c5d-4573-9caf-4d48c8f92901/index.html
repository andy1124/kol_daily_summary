三句話總結

  • SpaceX IPO題材持續佈局：馬斯克嗆Ryanair裝Starlink WiFi遭拒（燃油成本、短程航線不划算），廉價航空邏輯與Starlink服務定位不符，但Starlink仍是持續值得佈局的成長題材。
  • 台股市場廣度打開，傳產股開始動：工具機、上銀、全球傳動等過去被關稅壓抑的傳產股，因關稅談判進展開始補漲，市場從半導體獨強轉向更廣泛參與。
  • 台美脫鉤現象值得觀察：台股近期顯著強於納指，但歷史上最終會收斂（強的跌少或弱的補漲），目前台股是三市場中最健康、市場廣度最寬的。

投資觀點

  • 廉價航空vs Starlink的商業邏輯：極致低價壟斷市場的廉價航空（Ryanair等），客戶對服務品質不敏感，裝WiFi無法顯著提升票價或市占；Starlink的高端WiFi服務更適合傳統航空/長程路線。
  • 台股傳產佈局機會：過去因關稅（日韓競爭報價壓力）被壓抑的工具機、精密加工等，現在因關稅談判進展而反彈；代表市場開始定價關稅逆轉效果。
  • 台股相對強勢，但需謹慎脫鉤收斂：台美脫鉤如果反向修正（台跌補美漲），要有心理準備；但也有可能是台股基本面真的較強。
  • 魯特尼克的40%半導體在美目標：政治言語，短期內不可能實現，不需過度解讀；貝森特的思維與政策框架更值得追蹤。
  • 網路言論不等價：市場批評聲音多數來自非市場參與者，要先查清楚講話的人是誰再評估其觀點可信度。

SpaceX與衛星市場

馬斯克 vs 瑞安航空（Ryanair）

  • 事件：馬斯克建議Ryanair在機上安裝Starlink，遭Ryanair CEO Michael O'Leary拒絕
  • Ryanair的理由：
    • 機上天線增加重量與空氣阻力，估計每年多花1.5億美元燃油成本
    • 平均航程只有一小時，乘客不需要WiFi
  • 馬斯克反擊：波音737-800測試顯示燃油影響僅0.3%，威脅要把市場轉給其他航空
  • 主持人判斷：馬斯克挑錯對象；廉價航空客戶100%價格敏感，品牌忠誠度幾乎為零，裝WiFi無法帶來顯著競爭優勢
  • Starlink更適合的場景：有2-3家以上競爭的路線，且乘客（如投資人、需要在盤中時間出行者）願意為快速網路付溢價

  • 布局策略：提早佈局，收割IPO前後大量媒體版面帶動的FOMO資金

  • 意涵：不只是半導體/電子股能賺錢，傳產也開始有機會，市場廣度真正打開
