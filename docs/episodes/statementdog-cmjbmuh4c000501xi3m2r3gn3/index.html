495.【財經時事放大鏡】通電！排除AI資料中心的一切阻礙

節目：財報狗 Podcast
日期：2025-12-18

提及公司

  • 美股：NVIDIA、GE Vernova（GV，渦輪發電機/電力設備）、CoreWeave（AI雲端，交付遞延）、Nebius（AI基礎設施）、Tesla（Megapack）、Microsoft（微軟）
  • 台股：華城電機（電力設備/電網變壓器）、鴻海（進入資料中心建設市場）
  • 其他：（背景討論涉及 EIA 美國能源資訊署數據）

三句話總結

  • 電力才是AI最大瓶頸：微軟CEO坦言最大限制是電不是算力；美國電力新增量從10年均值10GW/年，暴增至2025年55GW、2026年規劃67GW；80%用電量集中在第三階段（GPU部署），而多數資料中心仍在第一、二階段。
  • NVIDIA召開閉門AI電力高峰會：只邀新創，目的是協助解決資料中心供電短缺；HVDC架構（效率提升45%）是NVIDIA力推的節電方向，凸顯電力攻擊已成下一波AI建設核心命題。
  • 投資輪動訊號：GPU內部「炒完」→電力設備「接棒」：CoWoS、水冷板、HBM等已完成一輪炒作；天然氣渦輪（GE Vernova訂單28年翻倍）、電網變壓器（華城）、太陽能等電力供應鏈成為新焦點。

投資觀點

  • 電力設備已進入加速確認期：GE Vernova表示訂單積壓（Backlog）從1000萬成長至2000萬，電氣化訂單直接翻倍，2829年訂單已滿；華城電機法說會透露某廠要直接Double產能，準備大賺這波需求。
  • 資料中心三階段電力分配：第一/二階段（土地+蓋廠+裝備）僅佔總電力需求20%；第三階段（GPU部署）佔80%，才是真正用電高峰。目前多數CSP還在第二階段，「電力壓力測試尚未開始」。
  • SMR（小型核電）短期看不到：供應商簽約多為2030-2035年後；成本超支、工期延誤歷史教訓多，CSP傾向簽PPA（電力採購協議）而非自建。
  • 天然氣渦輪是短中期最快補電方案：能源存量大（美國頁岩氣第一大出口國），開起來最快；主要瓶頸是渦輪產能有限（GE Vernova是關鍵供應商，與製造戰鬥機引擎的公司技術重疊）。

AI資料中心電力深度分析

需求爆發量化

  • 美國EIA數據：2010-2020年均新增電力約10GW/年。
  • 2023年：27GW；2024年：41GW；2025年：55GW；2026年規劃：67GW（約6倍於10年前均值）。
  • 試算：若台積電所有產能全數做成GB200並全部轉換算力，一年需電力達35GW以上；若美國佔全球AI算力70%，則AI貢獻電力需求約20-25GW/年，「非常緊」。

電力來源賽道分析

  • 天然氣渦輪（最快）：GE Vernova為主要供應商，訂單爆滿；美國頁岩氣充裕，接上電網較快，但渦輪產能本身是瓶頸。
  • 太陽能（次快）：蓋了就能發電，2025年預計佔美國新增電力近三成；不穩定性高，需搭配儲能。
  • 電網更新（重要基礎）：電網變壓器需求爆發（GE Vernova + 華城受益）；直流電纜/跨州高壓直流線路為新興討論方向。
  • SMR（最慢）：估值受電力需求波動，本質為長天期折現的公用事業型生意；2030年前無實質供電可能。

資料中心建設瓶頸

  • 「Powered Shell」（通電空殼）問題：CoreWeave、Nebius均坦承因工人短缺導致資料中心交付遞延。
  • 工人短缺類比台積電在美建廠：高技術電工人力嚴重不足，速度跟不上需求。
  • 鴻海等台廠已進入搶資料中心建設訂單市場，競爭更廣。

HVDC 節電邏輯

  • 傳統 AC→DC→DC 多次轉換，HVDC 減少轉換級數，效率提升約45%。
  • Data Center 用電量如此龐大，45%效率提升意義巨大；NVIDIA 力推HVDC，本質上是幫自己的晶片確保可以繼續出貨。

節目金句

  • 「微軟CEO說，他有多少電，他就能夠完成多少建設——最大的問題不是運算能力過剩，是電力。」
  • 「裡面炒完了，GPU互聯、CoWoS、水冷板、HBM，這些已經炒完了，現在大家開始 debate 外面的電力需求跟攻擊了。」
  • 「一年如果台積電的產能全部做成GP200，你這樣算一算一年要40幾GW——只有AI，美國佔70%，你就只有20幾GW，非常緊。」
  • 「電力壓力測試還沒開始——現在還在第二階段，80%的用電量是第三階段，GPU部署時才真正用到電。」
  • 「GE Vernova說他從1000多萬變2000多萬的Backlog，電力的電氣化訂單是直接翻倍——這個錢在你面前，你要不要加快產能來賺？」
  • 「SMR是緩不積極的，大家簽約都2030年之後——它本質上就是一個比較長折現的生意，有點像公用事業。」
  • 「NVIDIA沒有電就賣不了晶片——它一肩扛起全球股市，當然要出來搞這個電力問題。」
