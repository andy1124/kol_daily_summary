AI 重點摘要

提及公司

  • 美股：Bloom Energy、Micron（美光）、OpenAI、Anthropic、NVIDIA（輝達）、AMD、Meta、Microsoft
  • 台股：廠務股（未具名）、光通相關個股（未具名）

三句話總結

  • 記憶體利多不漲是警訊：美光財報超預期但股價未大漲，投資圈預期 EPS 喊至 70-84 美元各說各話，主委認為利多不漲代表資金開始思考退出或等待新的上修驅動力。
  • 需求驅動 vs 成本驅動漲價：當前供應鏈幾乎所有東西都在漲價，但投資時必須分辨是需求強勁帶動漲價（較佳），還是原物料成本轉嫁客戶（次等）；被動元件中，高階 MLCC 和 TELVR 電感屬於前者。
  • 「敢衝者贏」但也要警惕循環反轉：本輪 AI 供應鏈中，敢擴廠、敢接單的廠商是最大受益者，但 2021-2022 庫存逆轉的歷史提醒，一旦風向改變，囤最多的人可能摔最慘。

投資觀點

  • 資金已明確流向光通 / 光互連方向：主委認為 AI 資金從 OEM/ODM、ASIC 供應鏈，已轉向 interconnect/networking，可視為「AI 2.0」而非取代 AI。

    • 光通相關個股持續創高，估值高但一直有新消息上修
    • 高估值的風險：任何電話會議不如預期就可能大跌

  • 記憶體「Victoria's Secret 效應」：記憶體因報價暴漲 100% 而吸走了大量喜歡賭漲價的資金，其他同樣有漲價的類股（如被動元件、廠務股）資金被吸走，報酬反而不佳。

    • 若記憶體開始橫盤或轉弱，這些資金可能回流其他漲價類股

  • NAND 需求優於 DRAM：NAND 需求強勁（AI 使用者大量產生圖像/影片推動儲存需求），三大廠開會後傾向認為 NAND 漲價進度領先 DRAM；部分模組廠的低價庫存問題仍是市場疑慮。

  • 敢衝擴廠者是本輪贏家，但風險也最大：如同 OpenAI 大手筆押算力、台系模組廠敢囤積，膽量大者目前看起來是贏家；但 2021-2022 庫存循環崩盤的前車之鑑值得留意，差別在於「股票投資者隨時可以砍」。

SMR 與散熱需求分析

  • SMR（小型核反應爐）規模被嚴重低估：一般人以為 SMR 是小型設施，實際上一座 SMR 占地約兩個足球場，並非輕易解決資料中心電力的快速方案。
  • 散熱需求持續攀升：晶片越來越熱（TDP 越來越高），未來液冷（冷水/冰水）甚至兩相冷媒需求將持續成長；Immersion Cooling 等概念可能捲土重來。

被動元件供需分析

需求驅動 vs 成本驅動

  • 需求驅動（較佳選擇）：高階 MLCC（如車用、AI Server 用）、TELVR 電感——未來出現供需缺口，上修空間較佳
  • 成本驅動（次等選擇）：部分被動元件因使用到銀等原物料成本上升，被迫調漲報價；但這種漲價不代表需求強勁
  • 目前市場漲價標的眾多，但「市場麻木」現象：大家都知道有漲價，反而不容易在股票上反映

節目金句

  • 「我認識公司的誰之類的，所以大家對著之後有時候就會那種真相越辯越明的感覺。」
  • 「有些東西是怎麼講，就是你的防守範圍是有限的，你不一定會注意到。」

  • 「記憶體現在有點像說是市場裡面的那個 Victoria's Secret Model，大家太喜歡了，所以今天你漲那些報價，大家全部都往那邊衝。」
  • 「世界的任何萬物就是他最後面都是脫離不了循環，只是可能這一次的循環是比較大一點。」
