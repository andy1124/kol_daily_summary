日期：2026-06-01

提及公司

  • 美股：NVIDIA（輝達）、AMD（超微）、Microsoft（微軟）、Google、Meta、Amazon、Apple、Tesla、Micron（美光）、SK Hynix（SK海力士）、博通（Broadcom）、OpenAI、Anthropic、SpaceX
  • 台股：台積電、緯創

三句話總結

  • AI牛市第二階段：資金從晶片股擴散至伺服器、資料中心、軟體應用層，廢半單月漲幅達25%，軟體ETF（IGV）從4月低點反彈超30%。
  • MLCC缺貨潮：高盛點名被動元件MLCC將成下一個缺貨關鍵，AI伺服器一個機架需60萬顆，日韓大廠年產能成長僅一成，五年需求預估成長四倍。
  • 三缺制約泡沫：AI缺錢（四大CSP累積2,500億美元債務）、缺料（MLCC、記憶體）、缺電（資料中心預估2050年佔美商業用電22-30%），三缺使算力無法過剩，短期無泡沫破裂風險，但構成未來孤值回調理由。

投資觀點

  • AI牛市結構健康：本輪標普漲幅有限（今年以來約12%），廢半漲幅接近80%，系統性風險可控；大票不漲有助資金輪動至中小型廢半及軟體股。
    • 廢半一年漲幅達160%，今年已漲80%；軟體ETF IGV去年幾乎原地踏步，最近重返200日均線。
    • 軟體股空單部位達六年最高水準，一旦突破即觸發軋空。
  • 估值壓力可控但回調終將出現：缺料、缺電使算力無法過剩，不形成立即泡沫；但四大CSP資本支出持續加碼（Vera Rubin機架780萬美元，較GB300翻倍），成本壓力終將反映。
  • 廢半獲利成長確認無泡沫：前55家半導體廠2025年EPS年增148%；記憶體（Micron、SK Hynix的HBM）與晶片設計（NVIDIA、AMD）貢獻主力。
  • 資金輪動邏輯：NVIDIA市值登頂後難再上漲，資金釋出帶動其他廢半、微軟等相對低機的AI股補漲。

AI供需分析

AI缺錢

  • 全球四大CSP（超大型雲端服務商）累積超過2,500億美元債務，預估2030年全球AI基建投資逼近5兆美元。
  • 華爾街銀行風險額度接近上限，一邊放款給AI巨頭，一邊購買CDS避險。
  • 美國高收益債違約回收率從51%降至35%（疫情以來最低），銀行對中小企業放款趨保守。
  • SpaceX估值衝上9,650億美元，Anthropic、OpenAI等接續上市，下半年IPO搶錢壓力極大。

AI缺料（MLCC）

  • 高盛報告點名：MLCC（多層陶瓷電容）為繼GPU、記憶體後第三大漲價項目。
  • AI伺服器一個機架需60萬顆高階MLCC，用於管電、濾除電子雜訊。
  • 高階MLCC由日韓廠主導（村田製作所、三星電機），年產能成長約10%，五年需求預估成長400%（年複合34%），完全趕不上。
  • 整機成本持續上升：筆電、手機今年平均漲幅15-20%。

AI缺電

  • 資料中心建置量已超過商辦建置量；2050年預估佔美國商業建築用電22-30%。
  • 川普明確要求：資料中心旁直接設置天然氣廠，不得對當地電網造成負荷。
  • 黃仁勳在Computex被問及台灣2034年前供電無虞，連回兩句「Maybe」，引發外界解讀。
  • 區分：硬體製造階段的隱含能耗 vs 運行階段（GPU訓練推理散熱）的運營能耗，兩者皆將持續增長。
  • 緯創計畫在台北設第二座算力中心，因電力限制改遷台南沙崙。

台股行情

  • 台股5月漲5,800點，最高衝至44,954點；上週五爆出1.8兆元歷史天量。
  • 4月+5月合計漲7,200+5,800點，外資今年累計仍賣超約4,000億（5月已回補約2,000億）。
  • 加權指數6月1日站上45,000點（45,021點）。
  • 主要驅動：Computex黃仁勳人氣效應 + AI舊股（純台廠）受資金湧入。

軟體股軋空行情

  • IGV（軟體ETF）從4月低點反彈超30%，重回200日均線，近10個交易日表現超越費半。
  • 空單部位達近6年最高，突破即觸發軋空。
  • 1-2月「代理人AI末日論」（AI跳過App介面直接語音操作）引發軟體股恐慌已逐步修正。

AI倫理與法律監管

  • 教宗批評：AI快速進入醫療、金融、教育、政府服務，形成責任模糊狀態；呼籲軍事AI決策必須保留人類責任鏈。
  • 北韓公開測試AI引導巡弋飛彈，鎖定南韓邊境100公里內目標，AI軍事化已擴散至小型國家。
  • 監管真空形成：若AI做出錯誤醫療建議或評估福利資格錯誤，責任歸屬至今未有機制。
