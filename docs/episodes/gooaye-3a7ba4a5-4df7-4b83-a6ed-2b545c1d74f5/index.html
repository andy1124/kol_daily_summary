AI 重點摘要

  • 台股：國巨（Yageo）、三星電子（韓國）

三句話總結

  • CPU 成為 Agentic AI 時代新焦點：NVIDIA 入股 RISC-V 架構公司 SiFi，AWS Graviton CPU 產能被兩家客戶全包詢問卻遭拒絕，CPU 供不應求態勢逐漸浮現，帶動 CPU 相關股票走強。
  • 被動元件開始出現缺貨跡象：鋁電容、SP Cap 已嚴重缺貨，MLCC 接近緊線，下週三國巨法說會預期全面正面，類比 2017 年手機/電動車主軸帶動缺貨潮，現在主軸換成 AI 伺服器。
  • 光通供應鏈出現「貴人」效應：美系大廠因自建產能不足，委託與本業無關的台廠（如 SMT 打件廠、LED 廠）代工光通元件，帶設備進場生產，台股已有 4 家廠商受惠，形成新一波題材。

投資觀點

  • 選擇比努力重要：Q1 因持有過多「拖油瓶」（軟體、部分巨頭），即便有光通部位仍拖累表現；Q2 調整資金配置、聚焦 CPU + 光通相關後，帳戶快速回正並攻新高
  • 多頭要很多頭：美股現在除了光通（Broadcom），開始有 CPU 等新族群走出來，這是健康多頭的必要條件；台股類股輪動是最好示範
  • 走勢好的東西才有興趣：NVIDIA 雖好，但沒過新高暫時不感興趣；基本面是前提，技術面是加分——看到基本面好 + 新高突破才加碼
  • 軟體股（CrowdStrike 等）仍受 AI Doomsday 情緒壓制：每次 Anthropic 釋出誇大宣傳，市場就去打壓 SaaS/軟體整合商；資金暫時不在這，等情緒冷靜後再找介入時機

CPU 行情分析

NVIDIA 入股 SiFi

  • SiFi 是 RISC-V 架構的 CPU 新創公司
  • 產業圈有在討論：Agentic AI 時代除了 GPU 算力，需要大量 CPU 作為「調度教練」，RISC-V 特化版 CPU 有可能被重新關注

Amazon Graviton CPU 供不應求

  • 有兩家客戶要求全包 AWS Graviton CPU 算力，AWS 直接說「我們自己都不夠用」
  • 隱含訊號：
    • 亞馬遜後續必然加大 CPU 投片與生產
    • 這些客戶將轉向 Intel、AMD 等通用 CPU 廠
  • 觀察方向：從 Server 端 CPU → 可能延伸至 Custom Made ASIC CPU → 最後觀察是否影響消費端（手機/PC）機會

供應鏈受益鏈

台積電（先進製程）→ M-Core、SPEO（先進封裝）→ 其他相關後段廠，需求持續拉動

被動元件行情

  • 鋁電容（SP Cap）：已嚴重缺貨
  • MLCC：接近緊線；高階 AI 應用 MLCC 最缺的是春田（Murata）、Samco（TDK）等高階廠商產品
  • 下週三（國巨法說會）為催化劑，預期全面正面，持續漲價並帶動旗下積美（Kimetsu？）電容漲價
  • 類比邏輯：2017 年手機/電動車主軸帶動缺貨 → 2024 年 AI 伺服器成主軸，邏輯相同；可能的觸發事件類比 2024 年的 Sora/SanDisk 漲價訊號

光通代工新趨勢（台股）

背景

  • 美系光通大廠需求暴增，自建產能曠日廢時（環評/土地取得困難）
  • 轉而委託台灣 有閒置 SMT 產線的廠商（LED 廠、網通廠等），大廠自帶設備進場代工

現況

  • 台股已有 4 家廠商進入此模式，毛利約 10%（低但正收益），且訂單量大
  • 潛在標的識別方式：找有 SMT 打件產線但稼動率偏低的廠商 → 調查是否有新客戶進駐 → 確認後介入
  • 風險：大客戶可以給也可以抽走；驗證階段股價可能先炒高後崩落，需謹慎

Anthropic Claude Methos 評論

  • Anthropic 推出 Claude Methos，但僅開放少數策略夥伴，宣稱「東西太強大不能公開」
  • 謝孟恭（主持人）認為：真實原因是算力不足，公開後大量用戶上線會讓模型變「笨」（響應品質下降）
  • 每次 AI 大廠誇大宣傳，都會導致 CrowdStrike、Palo Alto 等 SaaS/整合商類股被市場打壓
  • 預計等 Anthropic 確保算力充足後才會正式放出；與 2019 年 OpenAI「東西太危險」的話術如出一轍

操作策略

  • CPU 相關：持續持有，並觀察是否有消費端（2C）機會外溢
  • 被動元件：已有部位但未放大，等待法說會或缺貨事件做催化
  • 光通/Broadcom：走勢最強，持續觀察
  • 軟體股（CrowdStrike 等）：作為長期持股繼續持有，不加碼，等資金回來再注意
  • Tesla：長期持股，但 Terra FSD 計畫形成大坑，短期不確定性高，依走勢調節而非滿倉
