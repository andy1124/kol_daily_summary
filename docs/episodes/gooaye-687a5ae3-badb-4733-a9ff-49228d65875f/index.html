三句話總結

  • 台股多頭型態確立，目標看 4 萬以上：台積電訂單未被取消、法說維持 Guidance 上緣，指數跟著台積電算，破 4 萬點是基本心理準備；多頭回調都是買點。
  • 被動元件全面開噴，電阻漲價信現身：日股被動元件已「No Brain」射出，美股 Vishay、VPG、Eton 接力大漲；台股龍頭國巨也在歷史新高區，僅腳步稍慢；最新訊號是電阻漲價信，先前市場預期缺貨的是電容與電感。
  • CPU 是房間裡的大象：Agentic AI 帶動 CPU 需求，主持人認為市場尚未充分估算其缺貨規模，可能比記憶體更恐怖（記憶體未斷貨，CPU 則有可能買不到），但漲價利潤落在哪個環節尚不確定。

投資觀點

  • 動能 > 價值：廠務股（長週期基礎設施相關）算得出便宜但跑得慢；市場偏愛動能股，資金都去追現在就在噴的東西，多頭格局下不需要逆勢等待低估值。
  • 國際村思維：台股的某個族群跑得慢，就直接去買日股或美股的同族群；像是記憶體報價漲得漂亮但台股記憶體股車上已太多人，反而外資在漲。
  • 資金輪動路徑預測：光通 → 記憶體 → 缺貨漲價故事（ABF 載板、材料、Q 布、被動元件）；猜對資金下一站的方向比猜個股更重要。
  • VR / Rubin 備料剛開始：BOM 表才剛公布，廠商尚在開始備料，但部分供應商稼動率已接近 90-100%，若 Rubin 正式拉貨將是明顯缺口。

被動元件分析

漲價訊號時間序列

  • 電容、電感：早已知道很缺，市場已充分討論。
  • 電阻（最新）：周五收到漲價信，超出預期；原因可能是金屬原物料漲價成本轉嫁，也可能是提前反映供需，稼動率是關鍵觀察指標。
  • 類比記憶體前例：記憶體早期稼動率未滿時多數人不看好缺貨，但稼動率從 90% 跳到 100% 是瞬間的，被動元件可能重演同樣劇本。

個股觀察

  • 國巨：台股龍頭，還原權息後也在歷史新高，只是爬坡速度比日美 player 慢。
  • V-Share（Vishay）：週四大噴。
  • VPG（Vishay Precision）：接力往上，準備突破。
  • Eton：整個拉出去，屬意外驚喜型。
  • 漢唐：在本波行情中挑選時間點表態，值得關注但未深入討論。

CPU 缺貨主題

  • DG Times 報導：CPU 缺貨程度可能比記憶體更恐怖——記憶體是沒有斷貨，CPU 是可能買不到。
  • AMD、Intel：目前漲幅在 20% 以下，若缺貨劇本成立還有上拉空間。
  • 供應鏈漏洞：業者在備 GPU / AI Sync 相關料號，但未充分備 CPU；若客戶突然帶著 CPU 需求進來要組 General Purpose Server，會產生搶料效應，帶動 Socket 及相關元件再度漲價。
  • 漲價落點未定：漲價效益究竟留在 CPU 製造商、通路商還是元件商，尚難確定。

AI 應用的反思：Token Maxing 泡沫風險

  • Uber 案例：開年四個月就用光全年 AI Token 額度，市場一片看多解讀，但主持人認為關鍵在財務數字能否對應——燒更多 Token ≠ 產出更多價值。
  • 生產力通膨：每個人都在 Token Maxing，競爭者也在用 AI，最終真正受益的是賣算力的 Server 廠，而不是使用者。
  • Result-Based vs. Cost-Based：軟體股中能撐住股價的都是 Result-Based（AI 直接帶來毛利/淨利改善）；只是形式上燒 Token 而無法 Ship 產品的公司，估值遲早會被市場重新檢視。
  • 改名 AI 潮後遺症：和過去加密貨幣、NFT 改名一樣，AI 泡沫的破滅信號是算力（Token）過剩；現在算力永遠不夠，但那些只吹不做的公司終究會被要求端出產品。
  • Apple 的潛在優勢：相較其他 CSP 不需要投入大量 AI 資本支出，現金流強勁，在 Token Maxing 轉向 Result-Based 評估的環境下，可能反而是市場偏愛的目標。

台積電法說解讀

  • Q1 Guidance 維持上緣：資本支出未上調，但在上緣位置本身就是信心表態；主持人預期未來某個時間點會看到資本支出正式上修。
  • 廠務股挑時間點出來：法說後廠務股反而有資金關注，可能原因是台積電沒有爆炸性上修，部分資金去找被低估的長週期標的。
