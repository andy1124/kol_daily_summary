# 三句話總結

- **荷姆茲海峽解封進展**：伊朗允許非美籍船隻通行，油價一度回落至88美元，川普喊話「永不持有荷姆茲」作為停火條件。
- **私人信貸流動性危機**：私募信貸開始限制贖回，科技巨頭為搶在窗口關閉前融資，OpenAI向私募基金開出17.5%最低回報率顯示資金吃緊。
- **市場底部邏輯**：短線由伊朗停火新聞驅動劇烈波動，真正中長期風險在私人信貸流動性崩潰；確認停火三條件尚未同時成立前，均屬情緒催化非趨勢訊號。

# 投資觀點

- **三項真正確認停火條件**：①伊朗公開承認停火條件 ②實質停火進展 ③荷姆茲海峽恢復正常通行——三者未同時成立前，油價與股市的反彈是短線情緒，而非趨勢反轉。
- **新聞驅動危機時代**：一句話可讓油價瞬間翻盤，避險成本飆升（標普1個月5%跌幅保護已不便宜），同時市場常在短時間大幅翻轉導致避險失效。
- **長期視角**：中東戰事終將落幕，真正挑戰是私人信貸在高利率下的脆弱性與科技股估值壓縮；AI是否能在戰後支撐股市高位仍是核心問題。
  * 根據過往歷史，戰爭（兩伊戰爭、科威特、利比亞）通常是石油產量瞬間暴跌後數年內逐步恢復，非永久結構破壞。
- **農業通膨外溢**：中東戰事帶動尿素價格短期漲20%，美國農民單季成本增加20萬美元；紅州農業州同時是川普最大基本盤，政治壓力加劇。

# 中東局勢與油價

## 荷姆茲停火談判

- 伊朗官方允許非美籍船隻通行荷姆茲，但未確認已與川普達協議；市場先行反映樂觀，油價回落。
- 川普受盟友壓力釋出和緩信號，認為「先談再說，談不成再打更合理」；82空降師已到位，6萬名美軍在中東，邊打邊談雙軌持續。
- 3月底前若停火未成，3月CPI確定受衝擊，4月至5月通膨數據將浮現，聯準會無降息空間。

## 能源結構缺口

- 全球1,600萬桶/天石油卡住，4月即便政策介入仍有逾1,000萬桶缺口。
- 卡達供應全球約30%氦氣（半導體製程冷卻不可替代），若設施受損，晶圓廠一季緩衝後將面臨結構性斷鏈。
- **受益方**：美國LNG商、澳洲天然氣商、美國工業氣體商（氦氣需求2025年1.76億方米→2035年3.2億，半導體爆發推動）。

# AI資金缺口與私人信貸風險

- **OpenAI**向私募基金開出17.5%最低回報——優質公司此舉顯示資金壓力大；同時關閉影片生成產品、終止11億美元迪士尼合作案，大幅收縮戰線。
- **SpaceX**：IPO規模從500億提升至750億，估值目標1.75兆，時程加速；OpenAI IPO可能提前至2026 Q3。
- **Glasswater**等私募信貸旗艦基金贖回量已大於流入量，被迫出售持股形成AI科技股賣壓。
- 25至30年AI資金缺口約3兆美元，私募信貸占8,000億；一旦私募信貸抽手，科技巨頭融資面臨重大障礙。

# 川普政治成本分析

- 油價上漲最重的地區恰是2024年投給川普的紅州（德州、佛州、北卡），支持者直接承受能源成本衝擊。
- 川普強支持率（鐵粉）開始鬆動，轉向「普通支持」；獨立選民支持度持續下滑。
- 移民執法副作用：建築業、農業、餐飲業合法勞工不敢上班，供應鏈延誤；2025年出現數十年來首次美國移民淨移出。

# 操作策略

## 時間框架選擇

- **核心問題**：停火是4月、5月、7月還是11月結束？決定資金要分幾個月投入，而非該不該投。
- 買進股票的理由是AI基本面，賣出也只能是AI基本面鬆動；若買進理由是逢低回調，則年線不破就持續是好時機。
- **長線格局**：從疫情至今每次回調逢低買盤均強勁，本質是資金在不同資產間輪轉，只要確定長期趨勢不變，短期波動只是表層現象。
