# 台積電法說前瞻

## Capex 超預期的意義

- **結果**：Capex 約 56B 美元，市場估約 48B，超預期明顯；確認先進製程大擴產周期正式進入高速期。
- **N2 需求極強**：先鋒量產良率比 N3 好，客戶排隊（Apple、NVIDIA、AMD、Google 等都舉手），此前擔心「只有 Apple 才用」的邏輯已失效。
- **產能稀缺性持續**：美國形狀（AI Startup）連「一片」的 N2 試量產都在搶；全球算力競賽倒逼台積電全速擴產。

## 受惠族群

- **廠務工程**：最直觀、確定性最高；全球每個地方都要蓋，台灣廠務工程廠也有機會輸出海外（美國、日本、新加坡已有經驗）。
- **前段設備**：以美日大廠為主，但值得持續關注。
- **化學耗材（Specialty Chemicals）**：2nm 製程上的化學材料台廠有新切入機會，排碳需求（就近供應）也是推力；晶片越大、耗材用量越多。
- **後段封裝測試**：先進封裝（CoWoS、SoIC）台廠占優，台積電釋出部分後段產能，日月光等受益。

# MLCC 供需分析

## 高盛報告核心內容

- **AI Server 占 MLCC 市場比例**：目前約 10%，預估 2025-2030 年成長 4.3 倍、2025-2027 年成長 2.7 倍。
- **Murata 給量**：每台 AI Server 約 45 萬顆 MLCC（每片 AI 加速卡 1.5 萬至 2.5 萬顆 × 8 卡 + 其他板卡）；Murata 預估 AI 相關 MLCC 用量 Value 成長約 30%（量略低，但 ASP 上升帶動 Value 更多）。
- **供需兩端同步緊張**：領頭廠商（Murata、太陽誘電、Samsung Electro-Mechanics）利用率已超過 8-9 成，開始有客戶反映排單有點緊張。

## HVDC 架構帶動高壓 MLCC

- **機制**：AI Server 走向 HVDC（高壓直流）架構，每個 DC-to-DC 轉換節點旁邊都需要高壓 MLCC 穩定電壓波動；電壓越高、規格越高、用量越多。
- **台灣廠商機會**：領頭日商轉做車用/高端後，傳統電腦用 MLCC 釋出給台廠；台廠在 AI Data Center 中低端 MLCC 市場滲透率有機會提升。

## 矽電容（Silicon Capacitor）崛起

- **定義**：用半導體製程做的電容，體積更小（較矮），適合空間有限的 AI 板；單價較高但占板面小。
- **市場現況**：部分 AI Server 設計師開始轉向矽電容，但需與晶片廠（台積電等）合作製程；傳統 MLCC 廠商也有壓力。

# AI 外溢：成本推動 + 需求拉動

## PMIC 漲價

- **高端滿載外溢**：高端 PMIC（AI Server 用 HVDC DC-to-DC）由大廠吃滿，8 吋廠利用率回升，PMIC 全線開始漲價。
- **中國廠商回補**：中國廠商在 155 計畫驅動下拉動 8 吋利用率，進一步推升 PMIC 需求。
- **成本面支撐**：銅、銀等 MLCC/PMIC 原料持續高價，廠商有合理漲價空間。

## 美國電力需求與太陽能

- **AI Data Center 帶動電力需求激增**：美國新增電力來源：太陽能 27.5%、天然氣調峰 30%、再生能源 30%、風電 12.5%。
- **太陽能隨裝即用**：小型/邊緣資料中心啟動時，太陽能是最快可部署的電源，驅動聯合再生能源等太陽能廠股價大漲（一個月+44%）。
- **改裝引擎發電機新創**：美國有新創公司將飛機引擎改裝成天然氣發電機，專供小型資料中心快速啟用。
