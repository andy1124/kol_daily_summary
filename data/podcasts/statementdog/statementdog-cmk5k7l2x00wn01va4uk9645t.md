# 投資觀點

- **AI 商業化無需懷疑**：Minimax 的 TOKI 每用戶年付費從 2023 年約 6 美元升至 2024 年 8 美元（之後因競爭增加而略降），海螺 AI 2025 年前 9 個月每用戶付費高達 56 美元——顯示確有人願意為 AI 付費，且圖像/影音的付費意願更高。

- **文生影音比文字更有定價力**：海螺 AI ARPU 遠高於純文字 TOKI，但 Sora 式工具留存率接近零，說明影音生成仍以一次性娛樂為主，真正有商業閉環的是整合遊戲化體驗的模型。

- **台灣製造商優勢**：推論成本下滑 → 毛利空間受壓 → 更依賴高效算力硬體 → 台積電等硬體供應鏈的議價能力依然關鍵；成本能降這麼快，台灣硬體廠功不可沒。

- **記憶體多層架構受惠**：AI 系統對 HBM → Context Memory → SSD 三層架構的需求，記憶體整體需求正當性無庸置疑，但具體受益標的需精細判斷（SSD 控制器炒作有誇大成分）。

# NVIDIA Context Memory 新架構

- 背景：AI 推理時 KV Cache 越來越大，現有的 V-RAM → HBM → SSD 三層架構中，SSD 距 GPU 太遠
- 新增層：在 HBM 與外部 SSD 之間，透過高速通訊協定部署鄰近 Switch 的 Context Memory（類似 HBF 概念）
- 市場反應誤解：部分投資人認為此舉等同 SSD 廠商大利多，MK 認為此 SSD 是通用規格，任何廠商皆可受惠，非特定廠商壟斷
- 真正正確的觀察：整體記憶體需求正當性提升（HBM → Context Memory → SSD 全線受惠），但非線性的「某特定廠商躺贏」

# 液冷散熱 45°C 水溫的澄清

- Jensen 在 Vera Rubin 發表會提到：只需 45°C 進水，不需要傳統冷卻設備（去了）
- 市場反應：液冷散熱族群短暫下跌
- 真實情況：
  * OCP 規範對液冷有 W17/27/32 不同等級，45°C 只是某一特定條件下的配置
  * 仍是水冷，只是進水溫度較高；不代表整個水冷散熱生態消失
  * MK 認為市場因斷章取義而錯殺，中型算力資料中心的水冷需求不變
