# 三句話總結

- **美光 DDR4 逆勢擴產**：美光在維吉尼亞州 Fab 6 重啟 DDR4 投產，支援車用與國防特殊規格需求，反映大廠退出消費性 DDR4 後整體供給偏緊的現實。
- **HBM 外包加速**：美光 HBM4 Base Die 委外台積電代工，HBM 量產進度超預期；控制 IC 外包（群聯、慧榮）趨勢明確，專業分工愈來愈清晰。
- **IPC/IPD 矽電容崛起**：台積電從2016年起已在封裝內嵌入矽電容，AI HPC 板面空間告急正在推動這類高整合度元件需求，被動元件廠與艾普正在搶進這個新市場。

# 投資觀點

- **DDR4 供需持續偏緊**：大廠轉向高階（HBM、DDR5），整體 DDR4 產能縮水，南亞科擴產計畫最快要到2028年，今年供不應求格局明確。
  * 現貨報價易被囤貨炒作，應觀察長期合約報價（Trendforce 等機構數據）。
  * 美光 DDR4 快速開出（預估年底），南亞科擴產較慢，市場敘事改變。
- **記憶體行業本益比天生偏低**：因景氣大循環特性，HBM 和高階 DDR5 是真正受惠 AI 的，DDR4 是外溢效應；要抓清楚不同產品的 tempo。
- **矽電容（IPC/IPD）值得追蹤**：規格比傳統 MLCC 更精密，單價較高，毛利較好；是「高科技 + AI 用在對的地方」的新元件，但有多種解決方案互為替代，供需較難精準掌握。

# 美光 DDR4 擴產事件

- 美光宣布 Fab 6（維吉尼亞州）開始投產 DDR4，主要支援車用與國防關鍵產業。
- DDR4 並未完全 EOL：部分特殊規格（車用、工業、網通）Controller 只支援 DDR4，換代成本過高，客戶不會主動升級。
- SSD 快取架構：NAND 搭配 DDR4 做 buffer，容量比約 1:2000，只要 SSD 大量出貨就有 DDR4 需求。
- 美光擴產訊息改變原本「大廠只做高階」的敘事，南亞科的收貨邏輯受到挑戰；但最終供需還是要看實際產能開出時程。

# 記憶體外包趨勢

- **控制 IC 外包**：NAND 大廠（美光等）利潤高的地方太多，開始集中資源；Controller 外包給群聯、慧榮，趨勢明確。
- **HBM Base Die 外包**：
  * 美光 HBM4 Base Die 委外台積電代工。
  * SK 海力士 Base Die 也持續委外台積電。
  * 三星自己做 Base Die，但曾出現問題導致速度落後。
  * Base Die 涉及 Logic 製程，記憶體大廠做不好（良率挑戰大），外包更有效率。
- **HBM4 規格演進**：下一代 Base Die 有機會超越 JEDEC 規範自訂設計，技術難度持續提升。
- 美光 HBM4 量產進度超乎預期快，股價過去一週漲26%；UBS 大幅調高目標價。

# IPC、IPD、IVR 元件解析

## 背景

- AI HPC 板面空間爆滿，元件必須往封裝內整合（CoWoS interposer 或 Substrate 內埋）。
- 台積電從2016年起在 InFO 等封裝中就有使用，手機板也早就在用（體積小、高度低）。
- 現在 AI 大板子也開始需要，IPC/IPD 市場正在快速成長。

## IPC / IPD（矽電容）

- 用矽製程製作的電容，可貼在晶片附近（Discrete 使用），或嵌入 Interposer/Substrate 內（InPackage）。
- 優點：體積更小、更薄，高頻特性優異（傳統 MLCC 在高頻下電容特性會消失）。
- 主要廠商：艾普（iP）、太陽誘電、muRata 等被動元件廠；製程用的是記憶體製程，偏簡單，與 DDR4 相同製程。
- 艾普近兩個月上漲，原因是切入 Intel EMIB 供應。

## IVR（整合式線性穩壓器）

- 電感 + 電容 + 電源元件的整合模組，負責垂直供電（越靠近用電端效率越高）。
- 台積電在自家 3D IC 封裝中已有使用，現在開始有外部元件廠要進入這個市場。

## 投資觀察重點

- 先觀察被動元件廠的 IPC/IPD 設計與量產推進狀況。
- 這類元件有多種替代方案，設計上不是唯一選擇，供需較難單一掌握。
- 市場目前太小，與 DDR4 整體設備市場相比量體還不夠大，但毛利好、成長方向確定。
