# 三句話總結

- **SaaS 估值重塑**：AI 拉高邊際成本、降低進入門檻，Rule of 40 失效，SaaS 估值應向重置成本靠攏，但擁有不可爬取私有數據的公司仍有護城河。
- **川普關稅被判違法**：IEEPA 對等關稅遭法院裁定違法，中國商品出現 3～6 個月低稅窗口期，航運與消費品（家具等）存在短線拉貨機會。
- **NVIDIA 法說超預期、股價反應平淡**：市場目光已轉向 Vera Rubin 出貨進度與三月 GTC 大會，GROQ 的 SRAM 整合推理架構（LPU）成為新焦點。

# 投資觀點

- **SaaS 本益比應全面下調**：AI 等同於燃料，凡能用電腦做的事都能被取代；邊際成本上升（AI API token 費用或自建算力）、超額利潤壓縮，SaaS 估值長期應趨向重置成本，ROE 與利潤率下降，轉為 cash cow 型態。
- **SaaS 中介平台不會消失，反而增加**：AI 需要更多 API 與資料彙整層（skills、data aggregator）；就像 Uber、Airbnb 出現後監管隨之而來，AI 同樣無法消除交易成本與監管障礙；Google SERP API 因 AI 需求反而爆量，印證中介只是換形態。
  * 金融機構因監管與問責壓力（需要「背鍋人」），核心系統短期不敢全面導入 AI
  * IBM 等舊系統依然存在，更換成本極高
- **資安股被錯殺**：AI 本身既是攻擊端也是防禦端，AI 平台部署在資安路口，資安需求不降反升，遭殺屬誤傷。
- **台灣市場 YTD +20%**：10 年前投資人以「缺乏軟體品牌」為由出走台股買美股；如今半導體供應鏈逆轉成主角，硬體估值持續走高，但集中度風險仍在。
- **中國 AI 春節爆發**：MiniMax、Seedance 等多家中國 AI 公司春節密集發佈新模型，競爭力不可輕視，值得持續追蹤。

# AI 對 SaaS 估值的衝擊分析

## 成本結構惡化

- **邊際成本上升**：過去 SaaS 邊際成本接近零，未來導入 AI 後不論打外部 API（按 token 計費）或自建模型（需算力），每單位成本顯著提高。
- **進入門檻下降**：AI 降低開發成本，競爭者大量湧入，超額利潤壓縮。
- **Rule of 40 失效**：成長率與利潤率雙雙承壓，過去 SaaS 高本益比的核心邏輯動搖。

## 哪些 SaaS 仍有護城河

- **私有數據壁壘**：AI 無法從公開網路爬取的數據（金融核心系統、法律文件等）是最大防禦優勢。
- **監管與問責壁壘**：金融機構不敢讓 AI 直接操作核心系統，傳統軟體供應商受益於客戶的風險規避心理。
- **SI（系統整合商）崛起**：AI 落地舊系統需要 SI 橋接；過去被視為 dirty 的「硬體綁軟體」模式，在 AI 時代反而可能形成黏著優勢。
- **資安平台剛需**：AI 是漏洞製造者，也是防禦者，AI 對抗 AI，資安平台角色不可取代。

# 川普關稅違憲：短線觀察

## 關稅結構背景

- **IEEPA 對等關稅被法院裁定違法**：此時此刻中國商品已不需繳交對等關稅，整體關稅稅率大幅下降。
- **後續加稅工具**：川普仍可動用 232（國家安全）、301（不公平貿易）、201（緊急進口）等條款重新課稅，調查程序約需 3～6 個月。

## 短線交易機會

- **目前關稅 < 未來關稅**：此窗口期是少數「現在買比之後買便宜」的時機，有動機提前拉貨。
- **受益標的**：航運（海運量增加）、轉移成本低的商品類（家具、包裝材料等特定品項）、中國製造商。
- **注意**：此為 3～6 個月的短線交易機會，非長線 thesis；若中國相關公司出現階段性業績好轉，需判斷是否為此因素造成。

# NVIDIA 法說與 GTC 展望

## 法說重點

- **全面超預期，股價反應冷淡**：市場對 NVIDIA 的標準極高，超預期被視為「理所當然」，缺乏額外驚喜。
- **Vera Rubin 延期 2～3 季**：屬「合理延期」，Blackwell 系列延期有先例，市場已消化此預期，延期責任指向特定供應鏈（「Alice」，詳見疑似辨識錯誤）。

## 市場新焦點

- **GTC 三月中**：市場傳聞將公佈 LPU（Language Processing Unit）相關架構，即 GROQ 的 SRAM 整合推理晶片，為新型推理加速方案。
- **光互連 vs 銅線**：短距離銅線仍具成本優勢，光互連尚未全面取代；股價已高漲的光互連標的進入深水區，需深入了解景氣波動。
- **零組件缺貨漲價**：電子原物料延續缺貨漲價趨勢。
- **市場關注重心轉移**：AI 合作消息（如 Meta x AMD）已是 comfort zone，股價反應趨弱；真正驅動股價的是新零組件缺貨、供應鏈變動等具體催化劑。
