# 三句話總結

- **先進製程DRAM供不應求加劇**：Q1三家大廠出貨量幾乎持平，全靠報價上漲驅動，三星季增93%、海力士與美光6~7成，三星罷工若成真將進一步收緊供給。
- **非HBM DRAM超越HBM表現**：推理與Agentic AI帶動通用伺服器需求爆發，DDR5供不應求程度超過HBM，三星非HBM比重高因此業績最亮眼。
- **下半年漲幅不易縮小**：先進製程供需缺口難以縮窄，成熟製程受消費性電子拖累但AI伺服器需求補位，整體報價上漲趨勢延續到下半年。

# 投資觀點

- **先進製程DRAM全年持多**：產能利用率已滿、廢本不足難以快速擴產，三星罷工更可能加劇缺口；主持人明確表示「Party沒結束，還會更興奮」。
  * 擴產需三至四季，且上游設備廠本身產能亦不足，瓶頸層層疊加
  * 三星一日罷工已使代工產量減少58%、記憶體減少18%，預告18天罷工影響巨大
- **成熟製程DRAM相對保守**：下半年偏向供不應求，但程度不如先進製程，消費性電子中低階衰退、長鑫存儲下半年放量皆構成壓力。
- **NAND Flash後勁強**：業者擴產表態較DRAM晚，產能釋放更慢，市場興奮程度後來居上，但玩家相對多（四至五家），擴產競爭風險存在。

# DRAM產業供需分析

## 先進製程（DDR5 / HBM）

- **Q1報價暴漲**：三星平均售價季增91~93%，美光64~67%，均由報價驅動，出貨量幾乎無成長。
- **非HBM崛起**：Agentic AI需要大量通用伺服器進行任務調度，CPU搭載DDR5而非HBM；三星HBM市佔落後，非HBM業務反而季增接近一倍，超越海力士與美光6~7成增幅。
- **HBM長約限制彈性**：HBM簽長約、大客戶溢價能力強（無人敢放棄NVIDIA），漲幅反不如現貨市場的非HBM靈活。

## 成熟製程（DDR4 / DDR5）

- **上半年強勁**：南亞科Q1營收成長63%、華邦電DDR5業務季增93%，表現亮眼。
- **下半年相對保守**：長鑫存儲（EY/EZ製程DDR5）預計下半年配合中國手機、PC廠商認證完成後開始放量，增加成熟製程供給。
- **AI伺服器SSD貢獻**：SanDisk預估今年企業級SSD需求成長75%，帶動搭載DRAM的成熟製程需求，但消費性電子衰退約抵消約10個百分點。

# 三星罷工風險

- **背景**：海力士員工平均獎金約1300萬台幣（營業利益10%分紅無上限），三星工會要求提高至15%且取消分紅上限。
- **談判破局**：5月21日計劃發動18天大罷工，三星半導體部門共7萬人，預計近七成員工參與。
- **產業影響**：罷工18天加上復工調教需兩至三週，實際產量損失超過一個月，先進製程供需缺口可能進一步擴大並帶動報價加速上漲。

# 操作策略

## 記憶體投資框架

- **先進製程**：持多到下半年，供需缺口短期難以縮窄，三星罷工更是額外催化劑；HBM以海力士、美光為主，非HBM三星受惠更明顯。
- **成熟製程**：上半年強、下半年謹慎，長鑫存儲放量與消費性衰退為主要壓力，報價成長幅度將不及先進製程。
- **記憶體模組廠與封測廠**：跟隨整體記憶體景氣，下半年仍有支撐但彈性相對受限。
