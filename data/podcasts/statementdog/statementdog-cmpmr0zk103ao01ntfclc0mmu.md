# 三句話總結

- **NAND Flash 需求爆發超出所有預期**：AI 推理的 KV Cache 卸載需求已從 HBM 外溢至 DRAM 再到 NAND Flash，Q1 2026 三星 NAND 業務季增一倍，各廠普遍季增 80-100%，速度遠超市場預期。
- **供給今明兩年難以追上**：NAND Flash 今年產能僅能靠製程推進增長約 17-20%，蓋新廠需 3-4 年，三星 NAND 部門罷工雪上加霜，2026 年下半年供不應求將進一步加劇。
- **TLC 需求壓倒 QLC，Kioxia 超高 IOPS SSD 與 NVIDIA 合作**：AI 推理系統要求速度，全部廠商法說會均強調 TLC 是今年主需求，QLC 普及延後至 2027 年，甚至出現更高速的 MLC/SLC 訴求（Super high IOPS SSD）。

- **NAND Flash 進入最佳供需格局**：需求爆發 + 供給結構性受限 + 消費性端預拉貨效應三重疊加
- **HDD 寡佔保障理性競爭**：Seagate + WD 合計市占 80%，兩家均強調長約模式、不激進擴產，價格有序上調
- **台灣 NAND 相關中游受惠明確**：惠龍、群聯因大廠退出消費性市場而市占率大幅提升；記憶體封測廠（力成、南茂、福懋科）受惠出貨量增長

# NAND Flash 需求爆發機制

## AI 推理 KV Cache 卸載（最核心驅動）
- 大型語言模型需記憶對話上下文（KV Cache），HBM 容量有限 → 卸載至 DRAM → DRAM 也不夠 → 進一步卸載至 NAND Flash（企業級 SSD）
- NVIDIA 已推出專用機櫃（CMX），專門處理 KV Cache 卸載，帶動整個產業跟進
- 價格倍數：NAND Flash 比 DRAM 便宜 10-100 倍 → 每次卸載下移，容量需求跳升 100 倍以上

## AI Agent（代理 AI）放大效應
- 代理 AI 可不眠不休自動執行任務，推理量是人工驅動的 100-1,000 倍
- 案例：某三人團隊使用代理 AI 一個月燒掉 OpenAI 費用約台幣 4,500 萬元（Fast Model 二倍計費）
- 代理 AI 產生的中間資料需暫存 → 過濾後有用資料存入 RAG server → RAG server 高度仰賴 NAND Flash SSD

## RAG（Retrieval-Augmented Generation）長期儲存需求
- 推理過程產生的有用資料存入知識庫（RAG server）
- RAG server 主要採用 NAND Flash SSD 儲存
- 正向循環：用得越多 → 資料越多 → KV Cache 需求越多 → 儲存需求越多

# NAND Flash 供給分析

## 今年供給增長受限（17-20%）
- 廠房空間不足，既有產線設備已滿，無法再從既有產線擴張
- 新建廠房需 3-4 年（如美光新加坡新廠預計 2028 年量產）
- 三大 DRAM 廠（三星、海力士、美光）優先把稀缺無塵室分配給 DRAM（單價與利潤率更高）
- 甚至有廠商把 NAND 廠房挪給 DRAM 生產

## TLC vs. QLC 制約供給
- AI 推理 KV Cache 要求傳輸速度 → 偏好 TLC（三位元），TLC 比 QLC 快但容量較小
- 若全用 QLC（四位元）可提升 1/3 產量，但 AI 客戶目前不接受
- 所有廠商（惠龍、凱霞、SanDisk、三星）均表示今年需求幾乎全是 TLC，QLC 普及推遲至 2027 年
- 凱霞推出 Super high IOPS SSD（採 MLC/SLC 技術），直接與 NVIDIA 合作，預計下半年量產

## 三星 NAND 罷工（2026 年 5 月）
- 政府調解破局，確定罷工，三星 NAND 部門（韓國產能約佔全球 40%）供給受衝擊
- 雪上加霜，進一步緊縮本已吃緊的供給

# 消費性 NAND 市場

## 淡季不淡
- 手機、PC 展望全面下調（雙位數衰退），但實際 Q1 消費性 NAND 季增仍有 80-100%
- 原因：下游客戶預期下半年漲價，提前拉貨（反正 NAND 價格只會更貴）
- 大廠把消費性 NAND 資源轉向 server，消費性供給也緊縮 → 消費性訂單轉向惠龍、群聯

## 注意事項
- 上半年提前拉貨效應若吸走太多需求，下半年消費性可能回落
- 但 AI server 端會持續吸走供給，整體報價仍有支撐

# HDD 產業補充（供需邏輯對照）

- Seagate 不擴產真正原因：HAMR 技術升級今年下半年量產，靠製程推進就能讓位元產量增長 50%+
- HDD 寡佔三家（Seagate、WD、Toshiba），前兩家合計市占 80%，均強調長約、不激進擴產
- 未來一年能見度超過四個季度，價格有序向上反應

# 疑似辨識錯誤

- 「NanFlash」/「NemFresh」/「Nanfraze」→ 建議：「NAND Flash」（Whisper 多種誤辨形式）
- 「吸結」→ 建議：「希捷」（Seagate，台灣慣用譯名，Whisper 誤辨）
- 「凱霞」→ 建議：「凱俠」（Kioxia，Whisper 辨識近似音）
- 「HANR」→ 建議：「HAMR」（Heat-Assisted Magnetic Recording，HDD 新技術，Whisper 誤辨 M 為 N）
- 「吉祥九一折」→ 建議：「即享九一折」（廣告段落，辨識誤差）
