# 三句話總結

- **NVIDIA 投資 Marvell：打 Broadcom 的策略佈局**：主要目的是把 ASIC 設計廠拉入 NVIDIA Fusion 生態系，讓 Broadcom 不再是唯一選擇，而非直接利多 Marvell 個股。
- **美股死貓跳、台股多頭格局未變**：NASDAQ 大幅反彈跳過季線；台股買太滿需趁反彈調節部分流動性；油價仍高，不加槓桿。
- **玻璃基板（GCS/TGV）2027-2028 放量**：市場提前關注，時間軸晚於 CPO 約 1-2 年；Intel 與 Broadcom 為先行者，台積電目前仍用 Glass Carrier 而非 Glass Substrate。

# 投資觀點

- **台股買太滿，趁反彈調節**：多頭格局下持續加碼強勢標的，但油價高位 + Vix 偏高，不宜再開槓桿；部分流動性要留給可能的「Depression Day」事件。

- **美股視為死貓跳、蟑螂人道走廊**：季線成為關鍵支撐；若反彈後維持在季線之上，可能視為短底；否則保持彈性，等待下一波加碼機會。

- **族群輪動觀察**：此波回檔後，觀察哪些族群反彈沒跟上（已作頭），哪些接近新高（新主流）；資金倒向新的強勢族群是操作核心。

- **地緣政治風險對沖**：中東戰爭持續，光通、記憶體、CPU 為主多頭；若戰爭升溫，可考慮鋰（旅）相關被動元件或原物料作為對沖。

# NVIDIA × Marvell 戰略解讀

## 投資動機分析

- **NVIDIA 現金太多，需要找地方放**：以最局外的角度看，財務投資本身沒有太多陰謀；重要的是戰略意圖。
- **核心目的：打 Broadcom**：
  * ASIC 設計過去只有 Broadcom 或 Marvell 可選；現在多了「NVIDIA Fusion Chiplet」選項
  * Broadcom 受競爭，但市場盤子夠大、不會死；Marvell 則多了 NVIDIA 背書

- **NVIDIA Fusion 架構**：
  * 小 Chiplet 可封裝入任意 XPU，提供 1.8 TBps 雙向頻寬
  * Vera CPU（ARM 架構，88 核）可為無 CPU 的 XPU 提供 CPU-based Solution
  * ConnectX NIC / Bluefield DPU / Quantum Switch / Spectrum 光通 Switch 全線支援

- **生態系軍火庫類比**：如同台積電——不論你是誰的 ASIC，NVIDIA 的互連、CPU、Switch、光通都可用。

## 對各方影響

- **Broadcom 最受衝擊**：本來是幾乎唯一的 ASIC 重量級設計夥伴，現在競爭加劇；但藍海夠大，不至於消亡。
- **Marvell 個股影響有限**：Jim Kramer 說要 All in Marvell，反而是警訊；這次投資更多是「宣示」而非極大利多。
- **StarFab / StarLabs（高速傳輸）**：與 Marvell 在不同層面，主攻 Host Side 記憶體池化；ASIC 混合資料中心衝擊有限，可共存。

# 玻璃基板（GCS / TGV）

## 發展時間軸

- **2026-2027**：Glass Carrier（玻璃載板）開始有量，台積電 CoWoS 目前使用此技術。
- **2027-2028**：GCS（Glass Core Substrate）開始放量，Intel + Broadcom 先行。
- **2029-2030 以後**：Glass Interposer（玻璃中介層）登場。

## 技術優勢

- **翹曲防制**：矽晶中介層尺寸受限，Intel E-MIB 改用玻璃基板解決大尺寸晶片（如 Google Humufish TPU V9）的 Warpage 問題。
- **TGV 製程**：雷射改質 + 化學蝕刻打穿孔，取代矽穿孔（TSV），成本理論上更低。

## 先行者

- **Intel**：EMIB 封裝 + Glass Core Substrate 已在日本展會展出；Google TPU V9（Humufish）是具體採用案例。
- **Broadcom**：在新加坡/日本 Shizuoka 合資產線，Tomahawk 8 預計採用玻璃（Tomahawk 6/7 仍在測試）。
- **台積電**：目前 CoWoS 只用 Glass Carrier，尚未採用 Glass Substrate。

## 投資角度

- 時間軸比 CPO 晚 1-2 年，但 CPO 股票也是提前 2-3 年就開始動；玻璃基板題材或許已開始進入市場視野。
- 不確定性高，觀察設備廠商是否集體轉強為確認訊號。

# 操作策略

## 部位管理

- **降低槓桿優先**：油價高位 + Vix 偏高，加槓桿的心性風險太大，容易引發非理性操作。
- **分批調節**：部位有層次，不在一個點位做全部決定；趁反彈賣出弱勢標的，保留流動性等待可能的大回調。
- **等待「Depression Day」事件**：若中東情勢進一步升溫（全面戰爭），市場先跌後漲機率高；手上要有現金才能加碼。

## 台股 vs 美股

- **台股**：整體仍強，但買太滿需趁反彈調節；跟大局保持彈性，不全面悲觀。
- **美股**：短底未確認；若反彈後再次破低，視為二次探底訊號，繼續減碼。

# QA

## NVIDIA Fusion 是否衝擊 StarFab 的 Retimer 需求

- **情境**：聽眾問 Marvell 的 Memory Pooling 解決方案是否影響 StarFab（Star Labs）的高速傳輸產品需求。
- **重點回答**：兩者在不同層面——Marvell Structure 是 Rack 內記憶體一致性（Host Side），StarFab 是 GPU Side 的 KV Cache Offload；CSP 混合資料中心影響有限，可共存。CXL 3.0 以上的 Rex Scale Fabric，兩家都有份；判斷正確，影響有限。
