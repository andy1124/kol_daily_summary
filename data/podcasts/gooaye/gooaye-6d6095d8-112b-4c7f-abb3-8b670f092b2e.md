# 三句話總結

- **Computex 敘事轉向**：今年展會最大變化是從「AI取代人」轉為「AI輔助人」，每個攤位都強調讓工作者更輕鬆、產品更具人格特質，而非走向無人化。
- **RTX Spark 機會**：NVIDIA RTX Spark 搭載 Unified Memory 架構，等效 5070 效能，可跑 Local Agentic AI 並大幅加速剪片修圖，主機說若遊戲表現強（如跑 GTA6），購買誘因將大幅提升。
- **市場短線震盪**：禮拜五夜盤台股最深摸負10%後拉回，禮拜一預計將是震盪洗盤日，融資與股息槓桿部位將被清洗，但產業基本面仍強。

# 投資觀點

- **AI 展會品質升級**：過去展會充斥機械狗、無意義 AI Gadget；今年展出項目全部是立即可用、真實影響工作與娛樂的東西，主機表示「每個東西看完就是 Shut up take my money」，對產業長期樂觀。
- **消費者體驗優先**：AI 不會「全自動取代創作者」，因為人格特質、個人風格是創作者的核心價值；AI 的角色是讓執行更快、讓人更有餘裕發揮個性，過度自動化反而是「加速自己滅亡」。
- **禮拜一短線壓力**：舒服久了就會有天罰，融資最大、股息槓最大的部位已開始被清洗，禮拜一預計多殺多，短期洗盤後再看新方向。
  * 部分股息高配息股已見開盤跳空低開後迅速拉回，意味被洗掉的是過度槓桿的人
  * 長線持有者無需過度緊張

# NVIDIA Computex 展會觀察

## Enterprise（TICC）

- **PowerRack**：已見展出，超大電容尺寸約原來5倍，ASP較過去高近100倍。
- **LPU**：多數封蓋，推估年出貨量約1萬至1.5萬 Racks；作為收購後全新路線，元件組成仍受市場關注。
  * 部分 QB（量子元件）已用於 LPU 內部
- **CPU Rack**：主攻 Agentic AI workflow；CPU 大量使用源於當前 Agentic workflow 尚未優化、模型撞牆消耗算力，但優化後整體使用量仍看增，因用戶基數持續擴大。

## 消費端（南港）

- **RTX Spark 亮點**
  - NYX（高規）售價預估 9–10 萬台幣，N1（低規）5–6 萬台幣。
  - 建議直上高規，原因是 Unified Memory 容量才是關鍵——需要塞入大模型、跑 Local Agentic AI 或高品質遊戲補幀，都需要大量記憶體。
  - 主機估算：若每月 API 支出可省下 1 萬元，10 個月即可回本，購買邏輯成立。
- **NVIDIA × Adobe 深度合作**
  - 展示使用 RTX Spark vs. 非 RTX 筆電跑 Premiere，速度可縮短至 1/5 至 1/7。
  - 自動識別鏡頭切換時間點，剪輯效率大幅提升。
  - Adobe 被市場預期「被 AI 淘汰」，但 NVIDIA 卻將其列為深度合作夥伴，值得重新評估。
- **RTX Spark × New York City 地貌渲染**：30–50 個 Block 可一瞬間 Render，無卡頓，展示 Unified Memory 的 rendering 能力。

## Asus ROG

- **精品化路線**：客製化鍵帽（盲盒）、24K 金背板、T1 聯名顯示卡，走精緻化路線應對消費端低迷。
- **AR 眼鏡**：主機親身試戴，畫面流暢度與畫質比肩 Apple Vision Pro，但重量僅一般眼鏡等級，認為「Future is now」，兩年內有機會做出非常完善的產品。
