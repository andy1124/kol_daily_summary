# OpenAI × AWS 合作傳聞

- **消息來源**：Information / Bloomberg 等媒體報導 OpenAI 正尋求 Amazon 融資。
- **預期架構**：類似 Anthropic 綁 Google 的模式——OpenAI 拿 AWS 融資，換取使用 AWS 訓練與推論伺服器（部分流量），雙方互相鎖定。
- **市場意義**：再次強化 AI 臥條聯盟（每家都需要對方），AI 生態系共同體更難崩解，但也代表若某一環節出問題，所有人一起爆炸。
- **操作建議**：AWS 相關股型態不佳，等股價實際反應後再決定是否追入；若連拉幾十%，再評估是否重新佈局。

# SpaceX IPO 與低軌衛星題材

## 題材邏輯

- SpaceX 估值極高，掛牌在即，帶動市場對低軌衛星供應鏈的想像空間擴大。
- 台股相關標的：**森達克**（Starlink BOM 占比高，股價已在右上角）、**滑通**（衛星版良率改善）、**藥滑**（良率相對較低但有潛力）。
- Starlink 主要市場在偏遠地區（Remote Areas），並非取代城市光纖，而是創造新市場（農村、戰場應用等），因此 TAM 低估的可能性高於低估。

## 操作觀察

- 當整個族群全面作動 + 美國有大型 IPO 加持，資金動能是真實的，估值可能先拉到 30 倍再下修 EPS 驗證。
- 需注意個別公司產品路線的變化：兩年前有些廠商被認為做 Router 占比大，但新一代設計可能已轉向；要跟進法說會或 Channel check 確認持續性。

## CPO 概念科普

- 雷射晶片（Laser Diode）：電→光，類比為手電筒開關，傳送 1010 訊號。
- Photo Detector：光→電，接收端將光訊號還原成電訊號。
- Transceiver Module（收發模組）：將雷射晶片 + Photo Detector + 控制 IC 封裝成一個可插拔模組。
- CPO（Co-Packaged Optics）：直接把雷射晶片封裝在交換晶片旁邊，縮短傳輸路徑、降低訊號衰減。
