# 三句話總結

- **超級CapEx週期確認**：四大超大型雲端業者合計資本支出接近6,700億美元（Amazon $200B、Google $185B、Meta $135B、Microsoft ~$150B），遠超市場預期，市場在空頭情緒下殺股反而製造買點。
- **Microsoft Maya加速推進**：微軟自研ASIC芯片Maya持續趕工，若成功放量將帶動相關供應鏈（機殼廠等），值得觀察是否能走出類似TPU題材的行情。
- **台灣硬體股與光通為市場相對強勢族群**：大盤震盪期間交換器廠、光通、散熱股逆勢走強，反映市場資金短線優先堆疊在CapEx受益的硬體端。

# 投資觀點

- **超大型雲端業者敢砸錢即代表「Open for Business」**：歷史上每次大雲端業者被批亂花錢（Cloud投資時期、AI初期），最終都帶來豐厚回報；現在市場又在重複同樣的批評。
  * Amazon/Google/Meta負責人回來親自操盤，不是白癡；「旁邊打嘴炮的分析師每個都最聰明」。
- **軟體（巨頭股）拉回是中長線買點，但技術面不破就不追**：目前巨頭走勢多為橫盤偏弱，策略是等碰到重要均線（240日線、720日線）且守住後再慢慢加碼。
- **硬體/供應鏈短線相對強**：台股交換器廠、光通（Lumentum）、散熱股在美股弱勢期間表現突出；CapEx數字確認後，硬體端的能見度比軟體端更高。
- **被動元件漲價僵固性形成利差空間**：鋼材等原物料價格下跌，但廠商已鎖定的漲價未必回調，存貨迭代後可能看到毛利改善機會。
- **市場在恐慌時才是加碼好時機**：當天台股早盤大跌，是難得的分批建倉時機；但若只是「慢慢跌一點一點跌」沒有明顯恐慌，則再等。

# 超大型雲端業者CapEx分析

## 四大業者數字

| 公司 | CapEx（年化） | 備注 |
|------|-------------|------|
| Amazon | $200B | 包含AI server、物流機器人、Kuiper低軌道衛星，非全部給NVIDIA |
| Google | $185B | 以TPU為主，自研比例高，能供應量僅市場需求的30至40% |
| Meta | $135B | 主要採購General Purpose GPU（NVIDIA/AMD） |
| Microsoft | ~$150B（$37.5B/季×4） | Maya ASIC加速推進中 |
| **合計** | **~$670B** | 遠高於市場預期 |

## 供應鏈注意事項

- **Amazon**：Trainium自研芯片占比不低，不能全部計入NVIDIA/AMD營收；Kuiper衛星占比可能不小，需下修AI server目標市場。
- **Google**：TPU能供應量僅為市場需求的約30至40%，仍大幅缺貨；相關供應鏈：Celestica（交換器/CLS）、Flex、Lumentum（OCS光學互連，採用MEMS方案，非Coherent L-Cost方案）。
- **Microsoft**：Maya ASIC接近趕工階段，若成功出貨將帶動機殼廠等供應鏈；過去幾年一直是「繞賽」狀態，尚未成功落地。
- **Meta**：目前尚無自研ASIC（Essex/ASIC）突破跡象，偏向採購General Purpose GPU。

# 光通互連供應鏈

- **Google OCS（光學電路交換）**：採用Lumentum的MEMS方案，非Coherent的L-Cost方案；Lumentum股價近期在大盤弱勢中逆勢上漲，技術面走強是驗證基本面的重要信號。
- **Celestica（CLS）**：Google交換器主要供應商之一，股價從$60多漲至$200+（分析師自述曾在$60賣出，錯失大漲）。
- **Flex**：Google供應鏈之一。

# 被動元件

- **漲價僵固性**：原物料（鋼材等）價格已回落，但廠商之前以原物料上漲為由調漲的價格不會自動回跌（早餐店效應），存貨迭代後毛利結構改善。
- **值得追蹤**：留意那些已漲價、原物料成本同步下滑的被動元件廠，毛利數字可能出現正面驚喜。

# 操作策略

## 當日台股大跌應對

- 台股早盤大跌（分析師睡過頭，略晚進場），仍在相對低位分批補倉。
- 目標：短線做反彈；若大盤無法持續創新高，這一批倉位可能維持帶狀橫盤策略。
- 海外放假期間持倉較輕，回台後將適度加大部位，特別偏向散熱相關標的。

## 美股策略

- 大型科技/巨頭股：等碰到240日或720日均線且守住後，慢慢小量加碼。
- 光通/交換器：Lumentum等逆勢強勢股可在盤整後技術面走強時考慮建倉。
- 散熱：若散熱股近期仍未表態，可考慮加大到此方向（CapEx數字支持）。
