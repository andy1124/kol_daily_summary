# 股癌 EP643 🦊
**發布日期**：2026-03-11

# 提及公司

- 美股：NVIDIA（Invidia）、Oracle、Micron
- 台股：台積電

# 三句話總結

- **伊朗戰爭引發台股大修正，主持人事後自評 70–80 分**：禮拜一看到原油跳漲而保守下手，部分部位賣在低點，但也因此避開最壞情境；核心邏輯是「VIX > 40 + 原油下行 = 買點」，本次只達成一半條件。
- **記憶體與台積電漲價消息陸續流出**：Micron 毛利率傳出可能突破市場 Forecast（>70% 以上），台積電下一季漲價幅度可能達 50%，但 overbooking 風險需同步監控。
- **NVIDIA LPU 第四代移往台積電（N3 + CoWoS-R）**：原本是三星題材，直下訊息顯示 V4 可能轉單台積電，帶動 LPU 從韓國題材變回台灣題材，相關供應鏈（前段 + 後段封裝）可能在 GTC 前後再度成為市場焦點。

# 投資觀點

- **多頭市場操作原則**：大跌時要有勇氣買，但需搭配指標（VIX 水位 + 原油方向）；伊朗最高領袖之子繼任、局勢不確定性未解之前，維持 20%+ 現金水位，拒絕槓桿。
  * 以 3 月 2 日股價為 benchmark：高於此價位 = 強勢股，優先加碼；低於此位 = 砍或觀望
- **LPU 題材邏輯**：NVIDIA 主體估值已在合理位置（4P 非常便宜），GTC 即便有消息但對 NVIDIA 本體影響有限；真正受益的是台灣供應鏈小型受益者，若蹭到 LPU 概念可帶一波。
- **CPX 放量低於預期**：市場對 CPX 採用熱度打得過滿，實際落地速度比預期慢；但長線仍看好，只是短期估值消化期拉長。

# 伊朗戰事與操作覆盤

## 事件回顧（2026 年 3 月初）

- 台股出現 2,000 點級別大跌，接近禮拜一為最低點。
- 伊朗最高領袖遭美軍斬首，但繼任者由其子出任（「王子復仇」邏輯讓主持人保持警覺）。
- 川普政府表示「禮拜二要炸一個大的」，增加週初不確定性。

## 操作覆盤

- **決策邏輯**：等 VIX > 40 且原油下行同時成立才積極入場。
- **實際狀況**：VIX 達標但原油跳漲到 100 美元以上，僅買了比規劃少的部位，同時砍掉部分熱門籌碼（擔心槓桿過重的人鎖跌停連鎖賣壓）。
- **結果**：部分賣在阿呆谷，但強勢股後來仍創新高，整體績效可接受，無法重來。
- **教訓**：下次應在覺得不對勁時更早降部位，而非等到事情爆發後才急砍。

# 記憶體市場

- **Micron 毛利率**：市場傳出可能突破目前 Forecast（約 70% 以上），遠超過街頭預估，是否兌現待財報確認。
- **台積電 CoWoS / HBM 相關漲價**：第一波漲幅已在進行中；下一季預計再漲 50%，但也有版本說只是「持續小漲、幅度收斂」——各方情報分歧，市場仍在試探。
- **overbooking 風險**：漲價引發大量囤貨，實際需求可能被高估；需觀察後續是否出現 Cancel 潮。

# NVIDIA LPU（Language Processing Unit）分析

## 版本進化

- LPU 競品（Groq）已推進至 V3，過去在三星代工。
- NVIDIA LPU V4：市場傳出轉往台積電（N3 製程 + CoWoS-R），後段封裝也在台積。
- 此轉換使 LPU 概念從「韓國（三星）題材」變為「台灣題材」。

## GTC 前後的市場邏輯

- GTC 是 NVIDIA 年度開發者大會，是 LPU 揭露的可能時間點。
- 主持人判斷：對 NVIDIA 股價本體影響有限（估值已合理），但對相關小型台灣供應鏈可帶波段。
- 前段（台積電 N3 CoWoS-R）+ 後段（台積電封裝廠 / 組裝廠）均可能受益。

## CPX 採用現況

- NVIDIA 另一產品線 CPX 的採用熱度低於早先預估，落地速度偏慢。
- 主持人判斷：不悲觀，只是市場估太快；數字會慢慢往上，但短期可能需繼續消化預期落差。

# 操作策略

## 現金與槓桿

- 持有現金 20%+ 為底線，拒絕在伊朗局勢未解前加槓桿。
- 原油維持在 80–90 美元以下為相對安全區間；若再度跳漲，代表新的地緣風險訊號，操作需更保守。

## 強弱篩選（以 3 月 2 日為基準）

- 高於 3/2 收盤價 = 強勢股，資金優先配置。
- 低於 3/2 = 觀望或砍倉候補。
- 遭受大盤震盪後仍在新高附近的標的，代表籌碼健康、信仰堅定，適合持有。

## 市場心理觀察

- 「只要比較就快樂不起來」：不去比較身邊做更好的人，比的是自己跟自己。
- 強調「知道為什麼做這個決定」比事後論對錯更重要。
