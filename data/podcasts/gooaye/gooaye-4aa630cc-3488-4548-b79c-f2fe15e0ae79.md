# NVIDIA架構深度分析

## Blackwell與新摩爾定律

- Transistor數×2，效能×10→Hooper Rack才是核心競爭力，硬體故事未結束。
- ICMS（Inference Context Memory Platform）：由Bluefield DPU管理的Storage Server，將KV Cache下放至Pseudo-SLC SSD；群聯Controller（TLC→模擬SLC）現身CES展示，群聯已進入NVIDIA Switch Tray。

## S-RAM Like題材澄清

- Groq使用S-RAM屬於資料中心（Data Center）應用。
- 華邦Cube、艾普VHM（Wave on Wave）可視為S-RAM Like，但客戶主要是手機端（H Device），**目前不在資料中心內**。
  * 不排除未來Apple/Samsung採用S-RAM Like技術降低對雲端依賴（AIPC夢醒或成真的時間點），但現在仍是H Device題材想象。
- Sudo S-RAM = D-RAM（以DRAM模擬SRAM），ETC（Embedded DRAM Tech）也屬此類，非真正SRAM。
