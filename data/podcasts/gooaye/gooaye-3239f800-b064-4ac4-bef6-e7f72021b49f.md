# 投資觀點

- **多頭行情下的倉位管理**：在強勢上漲行情中，應主動清理「長得慢」的部位，將資金集中到動能更強的標的；以機會成本思考部位配置而非以「賺賠」決定去留。
- **避免問標準答案**：技術面沒有固定時間軸，日線＋周線雙重確認優於追求精準天數；想避免時間軸雜訊可直接做「創新高」股票，最省力。
- **AI token 長期趨勢**：算力定價長期看跌，類比雲端運算演變軌跡；短期因 AI 需求爆發超過產能擴增速度而反彈，但一旦台積電等廠商持續上修產能，報價應回落。
- **AIS 是藍海**：新晶片設計玩家（如 Marvell 加入 Google 供應鏈）不會「零和」殺掉現有廠商，AI 市場在擴張中，博通、聯發科、Marvell 都能同時受益。

# 聯發科：Google AI 晶片合作深化

## V8（Zebrafish）& V9（Humufish）

- 聯發科負責 IO 晶片 + Memory Buffer，不做主晶片（主晶片由 Google 自設計）
- 概念：晶片設計代工，非單純晶圓製造代工，借助聯發科 SerDes IP 完成 Google 缺乏的部分
- 預期投片量：數百萬顆；下一代 Humufish 若順利，有機會拉升至 500–600 萬顆以上
- **3–4 年後 AI 貢獻有望超過整體 EPS 的 50%**，AI 純度高於高通

## 與 Broadcom 合作模式差異

- Broadcom（博通）：包辦晶片設計 + HBM 採購，拿走更高 ASP 與利潤
- 聯發科：分工設計，Google 保留核心控制權，聯發科利潤空間相對受壓但量大

## Humufish 旁的席位變化

- 原本 Humufish 旁還有另一顆 TPU，後來「消失」，疑由 Marvell 接入新席位（代號 MAROPAY）
- 若 Marvell 加入，聯發科下一代量可能進一步放大

# Google AI 供應鏈：Marvell 新加入

- 昨日外媒大量報導 Marvell 成為 Google 晶片設計新合作夥伴
- 已知兩顆案子：TIA（inference chip）+ memory processing unit（SRAM/HBM 整合）
- 合作模式：類似 Google × 聯發科，而非 Google × Broadcom（不完全外包）
- Marvell 強調記憶體整合能力，Google 可能意圖建立與 NVIDIA GPU 對標的 MPU 產品線
- 啟碁科技（6285）：SpaceX Starlink 訂單約占市值 50%，PCBA 業務是關鍵驅動
