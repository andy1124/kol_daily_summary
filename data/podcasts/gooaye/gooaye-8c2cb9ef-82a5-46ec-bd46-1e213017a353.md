# 投資觀點

- **別忽視龍頭**：市場資金習慣追小標的，但 TSMC 與 NVIDIA 的型態都在轉強，不排除後續補漲；龍頭也值得持有，不必只追題材股。
  * TSMC 最近型態明顯轉牆，搭配金管會開放投信改公開說明書，法人資金有望進場推升。
  * NVIDIA 也已突破前高向上。
- **被動元件出現橫盤整理**：熱門族群開始修正是健康的，至少在台灣法規（處置股）上，整理比一路噴更有利於後續操作；不代表多頭結束，斜率仍陡峭。
- **功率半導體（IDM）基本面改善**：ON、ADI 等功率元件廠基本面有明顯改善跡象，但目前缺乏全面性缺貨題材，需觀察電源管理需求是否進一步拉貨。
- **維持多頭但不積極追高**：目前指數連十字線都未跌破，操作上凹月線再做決策；若盤整後無法站回，才考慮減碼。

# Computex 2026 重點解讀

## NVIDIA（Jensen Huang 演講）

- **光通訊是必行方向**：從 plugable 到 CPO，NVIDIA 強調能用銅線就用銅線，但光的成長速率與未來發展持續拉大，資金可能再往光通訊方向移動。
- **Vera Rubin 已充分討論**：市場對 Rubin 已有充分定價，本次較新亮點是 **DGX Spark（N1X）**——同一晶片做三種 spec（高階筆電、初階筆電、車用），正式挑戰 AIPC 市場。
- **Total Solution 趨勢**：NVIDIA 論述從賣 GPU → 賣 Holosuite → 全 infra supplier，Google、Meta 等也都在賣 Total Solution；ASIC 與 GPU 已非對立，而是共存。

## Marvell

- **DSP 表現不錯，ASIC 仍在發展路上**：相比 MediaTek 或 Blockhunt，Marvell 的 ASIC 成熟度仍有段距離，此次噴漲更多源於黃仁勳加持。
- **估值過度反應要謹慎**：主持人已進過幾次 Marvell 但多次在 ASIC 題材上失手，此次靠光通訊 cue 才成功；建議觀察後續走勢是否過度 P/E 擴張。

# AIPC 第二波

- **Snowflake 員工觀點**：開源模型目前落後閉源模型約半年，當開源模型達到「人類需求 9000 分」的水準時，地端裝置即可滿足大多數使用需求，雲端費用大幅下降。
- **AIPC 換機潮機會**：若地端算力在 1–2 年成熟，AIPC 單價高、用料規格高（被動元件、晶片均升級），有機會 offset 傳統筆電/PC 銷售衰退，1 台 AIPC 約等於 1.5 台傳統消費性裝置的 ASP。
- **兩年前 Qualcomm 筆電先例**：第一批高通筆電當時算力不敷使用，為「Geek 玩具」；此次若算力真正到位，應用滲透率將加速。
- **不代表 Data Center 停滯**：地端算力崛起不是零和，高階推論、最前端研究仍需 Data Center，兩者可共同成長。

# 操作策略

- **軟蛋捏好、拿平均報酬**：主持人核心持倉為 CPU（光通訊相關）與被動元件；被動元件整理時不追，等月線支撐確認後再視情況加碼。
- **題材切換中找機會**：不一定要追強勢股，被大家短期丟棄但多頭排列的標的（如近期灰打的 Broadcom/Google 相關）反而是主持人關注的收購機會。
- **TSMC / NVIDIA 補漲邏輯**：市場正在把強題材資金轉回這兩檔，若後續投信資金進場，有機會再推一段。
